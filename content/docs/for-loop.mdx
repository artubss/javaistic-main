## Loop for em Java

Em programação de computadores, loops são usados para repetir um bloco de código. Por exemplo, se você quiser mostrar uma mensagem 100 vezes, em vez de digitar o mesmo código 100 vezes, você pode usar um loop.

Em Java, existem três tipos de loops:

- loop for
- loop while
- loop do...while

Este tutorial foca no loop for. Você aprenderá sobre os outros tipos de loops nos próximos tutoriais.

## Loop for em Java

O loop `for` em Java é usado para executar um bloco de código um certo número de vezes. A sintaxe do loop `for` é:

```java
for (expressaoInicial; expressaoTeste; expressaoAtualizacao) {
    // corpo do loop
}
```

Aqui,

1. A **expressaoInicial** inicializa e/ou declara variáveis e executa apenas uma vez.
2. A **condicao** é avaliada. Se a **condicao** for `true`, o corpo do loop `for` é executado.
3. A **expressaoAtualizacao** atualiza o valor da **expressaoInicial**.
4. A **condicao** é avaliada novamente. O processo continua até que a **condicao** seja `false`.

Para aprender mais sobre condições, visite [operadores relacionais Java](operators#3-java-relational-operators) e [operadores lógicos](operators#4-java-logical-operators).

## Funcionamento do loop for em Java

O loop for funciona da seguinte forma:

1. **Inicialização**: Executa uma vez no início
2. **Condição**: Verificada antes de cada iteração
3. **Corpo**: Executado se a condição for true
4. **Atualização**: Executada após cada iteração
5. **Repetição**: Volta para a verificação da condição

```
Inicialização → Condição → Corpo → Atualização → Repetir
```

### Exemplo 1: Exibir um Texto Cinco Vezes

#### Entrada

```java
// Programa para imprimir um texto 5 vezes

public class Main {
    public static void main(String[] args) {
        int n = 5;
        // loop for
        for (int i = 1; i <= n; ++i) {
            System.out.println("Java é divertido");
        }
    }
}
```

#### Saída

```plaintext
Java é divertido
Java é divertido
Java é divertido
Java é divertido
Java é divertido
```

Aqui está como este programa funciona.

| Iteração |     Variável     | Condição: i `<=` n |                           Ação                            |
| :------: | :--------------: | :----------------: | :-------------------------------------------------------: |
|    1ª    | `i = 1`, `n = 5` |       `true`       | `Java é divertido` é impresso `i` é aumentado para **2**. |
|    2ª    | `i = 2`, `n = 5` |       `true`       | `Java é divertido` é impresso `i` é aumentado para **3**. |
|    3ª    | `i = 3`, `n = 5` |       `true`       | `Java é divertido` é impresso `i` é aumentado para **4**. |
|    4ª    | `i = 4`, `n = 5` |       `true`       | `Java é divertido` é impresso `i` é aumentado para **5**. |
|    5ª    | `i = 5`, `n = 5` |       `true`       | `Java é divertido` é impresso `i` é aumentado para **6**. |
|    6ª    | `i = 6`, `n = 5` |      `false`       |                    O loop é terminado.                    |

## Variações do Loop for

### 1. Loop for com Decremento

```java
public class ForDecremento {
    public static void main(String[] args) {
        // Contagem regressiva
        for (int i = 10; i >= 1; i--) {
            System.out.println("Contagem: " + i);
        }
    }
}
```

### 2. Loop for com Incremento Diferente

```java
public class ForIncremento {
    public static void main(String[] args) {
        // Incremento de 2 em 2
        for (int i = 0; i <= 20; i += 2) {
            System.out.println("Número par: " + i);
        }
    }
}
```

### 3. Loop for com Múltiplas Variáveis

```java
public class ForMultiplasVariaveis {
    public static void main(String[] args) {
        // Loop com duas variáveis
        for (int i = 1, j = 10; i <= 5; i++, j--) {
            System.out.println("i = " + i + ", j = " + j);
        }
    }
}
```

### 4. Loop for Infinito

```java
public class ForInfinito {
    public static void main(String[] args) {
        // Loop infinito (cuidado!)
        for (;;) {
            System.out.println("Loop infinito");
            // Use break para sair
            break; // Remove esta linha para loop infinito real
        }
    }
}
```

## Exemplos Práticos

### Exemplo 1: Calculadora de Soma

```java
public class CalculadoraSoma {
    public static void main(String[] args) {
        int soma = 0;
        int n = 10;

        // Somar números de 1 a n
        for (int i = 1; i <= n; i++) {
            soma += i;
        }

        System.out.println("Soma dos números de 1 a " + n + ": " + soma);
    }
}
```

### Exemplo 2: Verificador de Números Primos

```java
public class VerificadorPrimos {
    public static void main(String[] args) {
        int numero = 17;
        boolean ehPrimo = true;

        // Verificar se é primo
        for (int i = 2; i <= Math.sqrt(numero); i++) {
            if (numero % i == 0) {
                ehPrimo = false;
                break;
            }
        }

        if (ehPrimo && numero > 1) {
            System.out.println(numero + " é primo");
        } else {
            System.out.println(numero + " não é primo");
        }
    }
}
```

### Exemplo 3: Gerador de Padrões

```java
public class GeradorPadroes {
    public static void main(String[] args) {
        int linhas = 5;

        // Padrão de asteriscos
        for (int i = 1; i <= linhas; i++) {
            for (int j = 1; j <= i; j++) {
                System.out.print("* ");
            }
            System.out.println();
        }
    }
}
```

### Exemplo 4: Calculadora de Fatorial

public class CalculadoraFatorial {

## Exercícios Práticos

### Exercício 1: Tabuada

**Objetivo**: Criar uma tabuada usando loop for.

**Enunciado**: Crie um programa que gere a tabuada de um número.

**Resolução**:

```java
public class Tabuada {
    public static void main(String[] args) {
        int numero = 7;

        System.out.println("Tabuada do " + numero + ":");
        System.out.println("==================");

        for (int i = 1; i <= 10; i++) {
            int resultado = numero * i;
            System.out.println(numero + " x " + i + " = " + resultado);
        }
    }
}
```

### Exercício 2: Série de Fibonacci

**Objetivo**: Gerar a série de Fibonacci usando loop for.

**Enunciado**: Crie um programa que gere os primeiros n números da série de Fibonacci.

**Resolução**:

```java
public class Fibonacci {
    public static void main(String[] args) {
        int n = 10;
        int primeiro = 0, segundo = 1;

        System.out.println("Série de Fibonacci (primeiros " + n + " números):");

        for (int i = 1; i <= n; i++) {
            System.out.print(primeiro + " ");

            int proximo = primeiro + segundo;
            primeiro = segundo;
            segundo = proximo;
        }
    }
}
```

### Exercício 3: Verificador de Palíndromo

**Objetivo**: Verificar se uma string é palíndromo usando loop for.

**Enunciado**: Crie um programa que verifique se uma string é palíndromo.

**Resolução**:

```java
public class VerificadorPalindromo {
    public static void main(String[] args) {
        String texto = "arara";
        boolean ehPalindromo = true;

        // Verificar se é palíndromo
        for (int i = 0; i < texto.length() / 2; i++) {
            if (texto.charAt(i) != texto.charAt(texto.length() - 1 - i)) {
                ehPalindromo = false;
                break;
            }
        }

        if (ehPalindromo) {
            System.out.println("'" + texto + "' é um palíndromo");
        } else {
            System.out.println("'" + texto + "' não é um palíndromo");
        }
    }
}
```

### Exercício 4: Calculadora de Média

**Objetivo**: Calcular a média de n números usando loop for.

**Enunciado**: Crie um programa que calcule a média de n números.

**Resolução**:

```java
public class CalculadoraMedia {
    public static void main(String[] args) {
        int[] numeros = {85, 92, 78, 96, 88};
        int soma = 0;

        // Calcular soma
        for (int i = 0; i < numeros.length; i++) {
            soma += numeros[i];
        }

        // Calcular média
        double media = (double) soma / numeros.length;

        System.out.println("Números: ");
        for (int i = 0; i < numeros.length; i++) {
            System.out.print(numeros[i] + " ");
        }
        System.out.println("\nMédia: " + media);
    }
}
```

## Conclusão

O loop `for` é uma ferramenta fundamental em Java para:

- **Iteração controlada**: Executar código um número específico de vezes
- **Processamento de dados**: Percorrer arrays e coleções
- **Cálculos repetitivos**: Realizar operações matemáticas iterativas
- **Geração de padrões**: Criar estruturas visuais e lógicas

**Boas práticas**:

- Use nomes de variáveis descritivos para contadores
- Evite modificar a variável de controle dentro do loop
- Use `break` para sair prematuramente quando necessário
- Considere usar `continue` para pular iterações
- Mantenha o corpo do loop simples e legível

Lembre-se de que o loop for é ideal quando você conhece o número de iterações ou pode determiná-lo antes do início do loop.
