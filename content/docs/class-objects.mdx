## Classes e Objetos em Java

Java é uma linguagem de programação orientada a objetos. O conceito central da abordagem orientada a objetos é dividir problemas complexos em objetos menores.

Um objeto é qualquer entidade que tem um estado e comportamento. Por exemplo, uma bicicleta é um objeto. Ela tem:

- **Estados**: parada, primeira marcha, etc.
- **Comportamentos**: frear, acelerar, etc.

Antes de aprendermos sobre objetos, vamos primeiro conhecer as classes em Java.

## Classe em Java

Uma classe é um modelo para o objeto. Antes de criarmos um objeto, primeiro precisamos definir a classe.

Podemos pensar na classe como um esboço (protótipo) de uma casa. Ela contém todos os detalhes sobre os andares, portas, janelas, etc. Com base nessas descrições construímos a casa. A casa é o objeto.

Como muitas casas podem ser feitas a partir da mesma descrição, podemos criar muitos objetos a partir de uma classe.

### Criar uma classe em Java

Podemos criar uma classe em Java usando a palavra-chave `class`. Por exemplo,

```java
class NomeDaClasse {
  // campos
  // métodos
}
```

Aqui, `campos` [(variáveis)](/docs/variables-literals) e [métodos](/docs/methods) representam o **estado** e **comportamento** do objeto respectivamente.

- campos são usados para armazenar dados
- métodos são usados para realizar algumas operações

Para nosso objeto `bicicleta`, podemos criar a classe como:

```java
class Bicicleta {

  // estado ou campo
  private int marcha = 5;

  // comportamento ou método
  public void frear() {
    System.out.println("Funcionamento do Freio");
  }
}
```

No exemplo acima, criamos uma classe chamada `Bicicleta`. Ela contém um campo chamado `marcha` e um método chamado `frear()`.

Aqui, `Bicicleta` é um protótipo. Agora, podemos criar qualquer número de bicicletas usando o protótipo. E todas as bicicletas compartilharão os campos e métodos do protótipo.

<Callout>

**Nota**: Usamos as palavras-chave `private` e `public`. Estas são conhecidas como modificadores de acesso. Para aprender mais, visite modificadores de acesso Java.

</Callout>

## Objetos em Java

Um objeto é chamado de instância de uma classe. Por exemplo, suponha que `Bicicleta` é uma classe, então `BicicletaMontanha`, `BicicletaEsporte`, `BicicletaTouring`, etc. podem ser considerados como objetos da classe.

### Criando um Objeto em Java

Aqui está como podemos criar um objeto de uma classe.

```java
nomeDaClasse objeto = new nomeDaClasse();

// para a classe Bicicleta
Bicicleta bicicletaEsporte = new Bicicleta();

Bicicleta bicicletaTouring = new Bicicleta();
```

Usamos a palavra-chave `new` junto com o construtor da classe para criar um objeto. Construtores são similares a métodos e têm o mesmo nome da classe. Por exemplo, `Bicicleta()` é o construtor da classe `Bicicleta`. Para aprender mais, visite [Construtores Java](/docs/constructors).

Aqui, `bicicletaEsporte` e `bicicletaTouring` são os nomes dos objetos. Podemos usá-los para acessar campos e métodos da classe.

Como você pode ver, criamos dois objetos da classe. Podemos criar múltiplos objetos de uma única classe em Java.

<Callout>

**Nota**: Campos e métodos de uma classe também são chamados de membros da classe.

</Callout>

## Acessar Membros de uma Classe

Podemos usar o nome dos objetos junto com o operador `.` para acessar membros de uma classe. Por exemplo,

```java
class Bicicleta {

  // campo da classe
  int marcha = 5;

  // método da classe
  public void frear() {
    System.out.println("Freando!");
  }
}

public class Main {
  public static void main(String[] args) {
    // criar objeto
    Bicicleta bicicleta1 = new Bicicleta();
    
    // acessar campo
    System.out.println("Marcha: " + bicicleta1.marcha);
    
    // chamar método
    bicicleta1.frear();
  }
}
```

## Exemplos Práticos

### Exemplo 1: Classe Carro

```java
class Carro {
    // Campos (atributos)
    private String marca;
    private String modelo;
    private int ano;
    private double velocidade;
    
    // Construtor
    public Carro(String marca, String modelo, int ano) {
        this.marca = marca;
        this.modelo = modelo;
        this.ano = ano;
        this.velocidade = 0.0;
    }
    
    // Métodos (comportamentos)
    public void acelerar(double quantidade) {
        velocidade += quantidade;
        System.out.println("Acelerando... Velocidade atual: " + velocidade + " km/h");
    }
    
    public void frear(double quantidade) {
        if (velocidade >= quantidade) {
            velocidade -= quantidade;
        } else {
            velocidade = 0;
        }
        System.out.println("Freando... Velocidade atual: " + velocidade + " km/h");
    }
    
    public void exibirInformacoes() {
        System.out.println("Marca: " + marca);
        System.out.println("Modelo: " + modelo);
        System.out.println("Ano: " + ano);
        System.out.println("Velocidade: " + velocidade + " km/h");
    }
    
    // Getters e Setters
    public String getMarca() {
        return marca;
    }
    
    public void setMarca(String marca) {
        this.marca = marca;
    }
    
    public double getVelocidade() {
        return velocidade;
    }
}

public class ExemploCarro {
    public static void main(String[] args) {
        // Criar objetos
        Carro carro1 = new Carro("Toyota", "Corolla", 2020);
        Carro carro2 = new Carro("Honda", "Civic", 2021);
        
        // Usar métodos
        carro1.acelerar(50);
        carro1.frear(20);
        carro1.exibirInformacoes();
        
        System.out.println();
        
        carro2.acelerar(80);
        carro2.exibirInformacoes();
    }
}
```

### Exemplo 2: Classe Estudante

```java
class Estudante {
    // Campos
    private String nome;
    private int idade;
    private String curso;
    private double[] notas;
    
    // Construtor
    public Estudante(String nome, int idade, String curso) {
        this.nome = nome;
        this.idade = idade;
        this.curso = curso;
        this.notas = new double[4]; // 4 disciplinas
    }
    
    // Métodos
    public void adicionarNota(int disciplina, double nota) {
        if (disciplina >= 0 && disciplina < notas.length) {
            notas[disciplina] = nota;
        }
    }
    
    public double calcularMedia() {
        double soma = 0;
        for (double nota : notas) {
            soma += nota;
        }
        return soma / notas.length;
    }
    
    public boolean estaAprovado() {
        return calcularMedia() >= 7.0;
    }
    
    public void exibirInformacoes() {
        System.out.println("Nome: " + nome);
        System.out.println("Idade: " + idade);
        System.out.println("Curso: " + curso);
        System.out.println("Média: " + calcularMedia());
        System.out.println("Status: " + (estaAprovado() ? "Aprovado" : "Reprovado"));
    }
}

public class ExemploEstudante {
    public static void main(String[] args) {
        // Criar estudantes
        Estudante estudante1 = new Estudante("João", 20, "Engenharia");
        Estudante estudante2 = new Estudante("Maria", 19, "Medicina");
        
        // Adicionar notas
        estudante1.adicionarNota(0, 8.5);
        estudante1.adicionarNota(1, 7.2);
        estudante1.adicionarNota(2, 9.0);
        estudante1.adicionarNota(3, 6.8);
        
        estudante2.adicionarNota(0, 9.5);
        estudante2.adicionarNota(1, 8.8);
        estudante2.adicionarNota(2, 9.2);
        estudante2.adicionarNota(3, 8.9);
        
        // Exibir informações
        estudante1.exibirInformacoes();
        System.out.println();
        estudante2.exibirInformacoes();
    }
}
```

### Exemplo 3: Classe Conta Bancária

```java
class ContaBancaria {
    // Campos
    private String titular;
    private String numeroConta;
    private double saldo;
    private String tipoConta;
    
    // Construtor
    public ContaBancaria(String titular, String numeroConta, String tipoConta) {
        this.titular = titular;
        this.numeroConta = numeroConta;
        this.tipoConta = tipoConta;
        this.saldo = 0.0;
    }
    
    // Métodos
    public void depositar(double valor) {
        if (valor > 0) {
            saldo += valor;
            System.out.println("Depósito de R$ " + valor + " realizado com sucesso.");
            System.out.println("Saldo atual: R$ " + saldo);
        } else {
            System.out.println("Valor inválido para depósito.");
        }
    }
    
    public boolean sacar(double valor) {
        if (valor > 0 && valor <= saldo) {
            saldo -= valor;
            System.out.println("Saque de R$ " + valor + " realizado com sucesso.");
            System.out.println("Saldo atual: R$ " + saldo);
            return true;
        } else {
            System.out.println("Saldo insuficiente ou valor inválido.");
            return false;
        }
    }
    
    public void transferir(ContaBancaria destino, double valor) {
        if (this.sacar(valor)) {
            destino.depositar(valor);
            System.out.println("Transferência de R$ " + valor + " realizada com sucesso.");
        }
    }
    
    public void exibirExtrato() {
        System.out.println("=== Extrato Bancário ===");
        System.out.println("Titular: " + titular);
        System.out.println("Número da Conta: " + numeroConta);
        System.out.println("Tipo de Conta: " + tipoConta);
        System.out.println("Saldo: R$ " + saldo);
    }
}

public class ExemploContaBancaria {
    public static void main(String[] args) {
        // Criar contas
        ContaBancaria conta1 = new ContaBancaria("João Silva", "12345-6", "Corrente");
        ContaBancaria conta2 = new ContaBancaria("Maria Santos", "67890-1", "Poupança");
        
        // Realizar operações
        conta1.depositar(1000.0);
        conta1.sacar(300.0);
        conta1.transferir(conta2, 200.0);
        
        System.out.println();
        conta1.exibirExtrato();
        System.out.println();
        conta2.exibirExtrato();
    }
}
```

### Exemplo 4: Classe Produto

```java
class Produto {
    // Campos
    private String nome;
    private String codigo;
    private double preco;
    private int quantidadeEstoque;
    
    // Construtor
    public Produto(String nome, String codigo, double preco, int quantidadeEstoque) {
        this.nome = nome;
        this.codigo = codigo;
        this.preco = preco;
        this.quantidadeEstoque = quantidadeEstoque;
    }
    
    // Métodos
    public void adicionarEstoque(int quantidade) {
        if (quantidade > 0) {
            quantidadeEstoque += quantidade;
            System.out.println(quantidade + " unidades adicionadas ao estoque.");
        }
    }
    
    public boolean vender(int quantidade) {
        if (quantidade > 0 && quantidade <= quantidadeEstoque) {
            quantidadeEstoque -= quantidade;
            System.out.println(quantidade + " unidades vendidas.");
            return true;
        } else {
            System.out.println("Quantidade insuficiente em estoque.");
            return false;
        }
    }
    
    public double calcularValorTotal() {
        return preco * quantidadeEstoque;
    }
    
    public boolean temEstoque() {
        return quantidadeEstoque > 0;
    }
    
    public void exibirInformacoes() {
        System.out.println("=== Informações do Produto ===");
        System.out.println("Nome: " + nome);
        System.out.println("Código: " + codigo);
        System.out.println("Preço: R$ " + preco);
        System.out.println("Quantidade em Estoque: " + quantidadeEstoque);
        System.out.println("Valor Total em Estoque: R$ " + calcularValorTotal());
        System.out.println("Tem Estoque: " + (temEstoque() ? "Sim" : "Não"));
    }
}

public class ExemploProduto {
    public static void main(String[] args) {
        // Criar produtos
        Produto produto1 = new Produto("Laptop", "LAP001", 2500.0, 10);
        Produto produto2 = new Produto("Mouse", "MOU001", 50.0, 50);
        
        // Realizar operações
        produto1.vender(3);
        produto1.adicionarEstoque(5);
        produto2.vender(10);
        
        System.out.println();
        produto1.exibirInformacoes();
        System.out.println();
        produto2.exibirInformacoes();
    }
}
```

### Exercício 1: Classe Retângulo
**Objetivo**: Criar uma classe para representar um retângulo.

**Enunciado**: Crie uma classe `Retangulo` com métodos para calcular área e perímetro.

**Resolução**:

```java
class Retangulo {
    private double largura;
    private double altura;
    
    public Retangulo(double largura, double altura) {
        this.largura = largura;
        this.altura = altura;
    }
    
    public double calcularArea() {
        return largura * altura;
    }
    
    public double calcularPerimetro() {
        return 2 * (largura + altura);
    }
    
    public boolean ehQuadrado() {
        return largura == altura;
    }
    
    public void exibirInformacoes() {
        System.out.println("Largura: " + largura);
        System.out.println("Altura: " + altura);
        System.out.println("Área: " + calcularArea());
        System.out.println("Perímetro: " + calcularPerimetro());
        System.out.println("É quadrado: " + (ehQuadrado() ? "Sim" : "Não"));
    }
}

public class TesteRetangulo {
    public static void main(String[] args) {
        Retangulo retangulo1 = new Retangulo(5.0, 3.0);
        Retangulo retangulo2 = new Retangulo(4.0, 4.0);
        
        System.out.println("Retângulo 1:");
        retangulo1.exibirInformacoes();
        
        System.out.println("\nRetângulo 2:");
        retangulo2.exibirInformacoes();
    }
}
```

### Exercício 2: Classe Calculadora
**Objetivo**: Criar uma classe calculadora com operações básicas.

**Enunciado**: Crie uma classe `Calculadora` com métodos para operações matemáticas.

**Resolução**:

```java
class Calculadora {
    private double resultado;
    
    public Calculadora() {
        this.resultado = 0.0;
    }
    
    public void somar(double valor) {
        resultado += valor;
        System.out.println("Soma: " + resultado);
    }
    
    public void subtrair(double valor) {
        resultado -= valor;
        System.out.println("Subtração: " + resultado);
    }
    
    public void multiplicar(double valor) {
        resultado *= valor;
        System.out.println("Multiplicação: " + resultado);
    }
    
    public void dividir(double valor) {
        if (valor != 0) {
            resultado /= valor;
            System.out.println("Divisão: " + resultado);
        } else {
            System.out.println("Erro: Divisão por zero!");
        }
    }
    
    public void limpar() {
        resultado = 0.0;
        System.out.println("Calculadora limpa. Resultado: " + resultado);
    }
    
    public double getResultado() {
        return resultado;
    }
}

public class TesteCalculadora {
    public static void main(String[] args) {
        Calculadora calc = new Calculadora();

        calc.multiplicar(5);
        calc.subtrair(15);
        calc.dividir(2);
        calc.limpar();
    }
}
```

### Exercício 3: Classe Livro
**Objetivo**: Criar uma classe para representar um livro.

**Enunciado**: Crie uma classe `Livro` com informações básicas e métodos para gerenciar o status.

**Resolução**:

```java
class Livro {
    private String titulo;
    private String autor;
    private int anoPublicacao;
    private boolean emprestado;
    private String genero;
    
    public Livro(String titulo, String autor, int anoPublicacao, String genero) {
        this.titulo = titulo;
        this.autor = autor;
        this.anoPublicacao = anoPublicacao;
        this.genero = genero;
        this.emprestado = false;
    }
    
    public void emprestar() {
        if (!emprestado) {
            emprestado = true;
            System.out.println("Livro '" + titulo + "' foi emprestado.");
        } else {
            System.out.println("Livro '" + titulo + "' já está emprestado.");
        }
    }
    
    public void devolver() {
        if (emprestado) {
            emprestado = false;
            System.out.println("Livro '" + titulo + "' foi devolvido.");
        } else {
            System.out.println("Livro '" + titulo + "' não estava emprestado.");
        }
    }
    
    public boolean estaDisponivel() {
        return !emprestado;
    }
    
    public void exibirInformacoes() {
        System.out.println("=== Informações do Livro ===");
        System.out.println("Título: " + titulo);
        System.out.println("Autor: " + autor);
        System.out.println("Ano de Publicação: " + anoPublicacao);
        System.out.println("Gênero: " + genero);
        System.out.println("Status: " + (estaDisponivel() ? "Disponível" : "Emprestado"));
    }
}

public class TesteLivro {
    public static void main(String[] args) {
        Livro livro1 = new Livro("O Senhor dos Anéis", "J.R.R. Tolkien", 1954, "Fantasia");
        Livro livro2 = new Livro("1984", "George Orwell", 1949, "Ficção Científica");
        
        livro1.exibirInformacoes();
        livro1.emprestar();
        livro1.emprestar();
        livro1.devolver();
        
        System.out.println();
        livro2.exibirInformacoes();
    }
}
```

### Exercício 4: Classe Pessoa
**Objetivo**: Criar uma classe para representar uma pessoa.

**Enunciado**: Crie uma classe `Pessoa` com informações pessoais e métodos para calcular idade.

**Resolução**:

```java
import java.time.LocalDate;
import java.time.Period;

class Pessoa {
    private String nome;
    private String cpf;
    private LocalDate dataNascimento;
    private String endereco;
    
    public Pessoa(String nome, String cpf, LocalDate dataNascimento, String endereco) {
        this.nome = nome;
        this.cpf = cpf;
        this.dataNascimento = dataNascimento;
        this.endereco = endereco;
    }
    
    public int calcularIdade() {
        LocalDate hoje = LocalDate.now();
        Period periodo = Period.between(dataNascimento, hoje);
        return periodo.getYears();
    }
    
    public boolean ehMaiorDeIdade() {
        return calcularIdade() >= 18;
    }
    
    public String obterCategoriaIdade() {
        int idade = calcularIdade();
        if (idade < 12) return "Criança";
        else if (idade < 18) return "Adolescente";
        else if (idade < 60) return "Adulto";
        else return "Idoso";
    }
    
    public void exibirInformacoes() {
        System.out.println("=== Informações da Pessoa ===");
        System.out.println("Nome: " + nome);
        System.out.println("CPF: " + cpf);
        System.out.println("Data de Nascimento: " + dataNascimento);
        System.out.println("Endereço: " + endereco);
        System.out.println("Idade: " + calcularIdade() + " anos");
        System.out.println("Categoria: " + obterCategoriaIdade());
        System.out.println("Maior de Idade: " + (ehMaiorDeIdade() ? "Sim" : "Não"));
    }
}

public class TestePessoa {
```

## Conclusão

Classes e objetos são fundamentais na programação orientada a objetos em Java:

- **Classes**: Modelos ou moldes para criar objetos
- **Objetos**: Instâncias de classes com estado e comportamento
- **Encapsulamento**: Controle de acesso aos dados através de modificadores
- **Reutilização**: Múltiplos objetos podem ser criados a partir de uma classe

**Boas práticas**:
- Use nomes descritivos para classes e objetos
- Encapsule dados usando modificadores de acesso apropriados
- Crie construtores para inicializar objetos
- Implemente métodos para representar comportamentos
- Use getters e setters para controlar acesso aos dados
- Mantenha classes coesas e com responsabilidades bem definidas

Lembre-se de que a programação orientada a objetos ajuda a organizar código de forma mais modular e reutilizável.
