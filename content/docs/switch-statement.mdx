## Instrução switch em Java

A instrução `switch` nos permite executar um bloco de código entre muitas alternativas.

A sintaxe da instrução `switch` em Java é:

```java
switch (expressao) {

  case valor1:
    // código
    break;

  case valor2:
    // código
    break;

  ...
  ...

  default:
    // instruções padrão
  }
```

## Como funciona a instrução switch-case?

A `expressao` é avaliada uma vez e comparada com os valores de cada case.

- Se `expressao` corresponder a `valor1`, o código de `case valor1` é executado. Da mesma forma, o código de `case valor2` é executado se `expressao` corresponder a `valor2`.
- Se não houver correspondência, o código do **case padrão** é executado.

<Callout>

**Nota**: O funcionamento da instrução switch-case é similar à [escada Java if...else...if](/docs/if-else-statement#3-java-if-else-if-statement). No entanto, a sintaxe da instrução `switch` é mais limpa e muito mais fácil de ler e escrever.

</Callout>

### Exemplo: Instrução Java switch

#### Entrada

```java
// Programa Java para verificar o tamanho
// usando a instrução switch...case

public class Main {
    public static void main(String[] args) {
        int numero = 44;
        String tamanho;

        // instrução switch para verificar tamanho
        switch (numero) {
            case 29:
                tamanho = "Pequeno";
                break;

            case 42:
                tamanho = "Médio";
                break;

            // corresponder ao valor da semana
            case 44:
                tamanho = "Grande";
                break;

            case 48:
                tamanho = "Extra Grande";
                break;

            default:
                tamanho = "Desconhecido";
                break;
        }
        System.out.println("Tamanho: " + tamanho);
    }
}
```

#### Saída:

```plaintext
Tamanho: Grande
```

No exemplo acima, usamos a instrução switch para encontrar o tamanho. Aqui, temos uma variável `numero`. A variável é comparada com o valor de cada instrução case.

Como o valor corresponde a **44**, o código de `case 44` é executado.

```java
tamanho = "Grande";
break;
```

Aqui, a variável `tamanho` é atribuída com o valor `Grande`.

## Tipos de Dados Suportados

A partir do Java 7, a instrução switch suporta:

- **Tipos primitivos**: `byte`, `short`, `char`, `int`
- **Tipos wrapper**: `Byte`, `Short`, `Character`, `Integer`
- **String**: A partir do Java 7
- **Enum**: A partir do Java 5

### Exemplo 1: Switch com String

```java
public class SwitchString {
    public static void main(String[] args) {
        String dia = "Segunda";
        String mensagem;

        switch (dia) {
            case "Segunda":
                mensagem = "Início da semana";
                break;
            case "Terça":
                mensagem = "Segundo dia";
                break;
            case "Quarta":
                mensagem = "Meio da semana";
                break;
            case "Quinta":
                mensagem = "Quase fim da semana";
                break;
            case "Sexta":
                mensagem = "Último dia útil";
                break;
            case "Sábado":
            case "Domingo":
                mensagem = "Fim de semana";
                break;
            default:
                mensagem = "Dia inválido";
                break;
        }

        System.out.println("Dia: " + dia);
        System.out.println("Mensagem: " + mensagem);
    }
}
```

## Exemplos Práticos

### Exemplo 1: Calculadora Simples

```java
public class CalculadoraSwitch {
    public static void main(String[] args) {
        double num1 = 10.0;
        double num2 = 5.0;
        char operacao = '+';
        double resultado = 0.0;

        switch (operacao) {
            case '+':
                resultado = num1 + num2;
                break;
            case '-':
                resultado = num1 - num2;
                break;
            case '*':
                resultado = num1 * num2;
                break;
            case '/':
                if (num2 != 0) {
                    resultado = num1 / num2;
                } else {
                    System.out.println("Erro: Divisão por zero!");
                    return;
                }
                break;
            case '%':
                resultado = num1 % num2;
                break;
            default:
                System.out.println("Operação inválida!");
                return;
        }

        System.out.println(num1 + " " + operacao + " " + num2 + " = " + resultado);
    }
}
```

### Exemplo 4: Conversor de Temperatura

```java
public class ConversorTemperatura {
    public static void main(String[] args) {
        String unidadeOrigem = "Celsius";
        String unidadeDestino = "Fahrenheit";
        double temperatura = 25.0;
        double resultado = 0.0;

        String combinacao = unidadeOrigem + "_" + unidadeDestino;

        switch (combinacao) {
            case "Celsius_Fahrenheit":
                resultado = (temperatura * 9/5) + 32;
                break;
            case "Fahrenheit_Celsius":
                resultado = (temperatura - 32) * 5/9;
                break;
            case "Celsius_Kelvin":
                resultado = temperatura + 273.15;
                break;
            case "Kelvin_Celsius":
                resultado = temperatura - 273.15;
                break;
            case "Fahrenheit_Kelvin":
                resultado = (temperatura - 32) * 5/9 + 273.15;
                break;
            case "Kelvin_Fahrenheit":
                resultado = (temperatura - 273.15) * 9/5 + 32;
                break;
            default:
                System.out.println("Conversão não suportada!");
                return;
        }

        System.out.printf("%.1f %s = %.1f %s\n",
                         temperatura, unidadeOrigem, resultado, unidadeDestino);
    }
}
```

### Exercício 1: Calculadora de IMC

**Objetivo**: Usar switch para classificar IMC.

**Enunciado**: Crie um programa que classifique o IMC usando switch.

**Resolução**:

```java
public class CalculadoraIMC {
    public static void main(String[] args) {
        double imc = 24.5;
        String classificacao;

        // Converter IMC para faixa para usar no switch
        int faixa = (int) (imc / 5); // Agrupa em faixas de 5

        switch (faixa) {
            case 0:
            case 1:
            case 2:
            case 3:
                classificacao = "Abaixo do peso";
                break;
            case 4:
                classificacao = "Peso normal";
                break;
            case 5:
                classificacao = "Sobrepeso";
                break;
            case 6:
                classificacao = "Obesidade grau 1";
                break;
            case 7:
                classificacao = "Obesidade grau 2";
                break;
            default:
                classificacao = "Obesidade grau 3";
                break;
        }

        System.out.printf("IMC: %.1f\n", imc);
        System.out.println("Classificação: " + classificacao);
```

### Exercício 3: Validador de CPF

**Objetivo**: Usar switch para validar diferentes aspectos do CPF.

**Enunciado**: Crie um programa que valide CPF usando switch.

**Resolução**:

```java
public class ValidadorCPF {
    public static void main(String[] args) {
        String cpf = "12345678909";
        String tipoValidacao = "comprimento";
        String resultado;

        switch (tipoValidacao) {
            case "comprimento":
                if (cpf.length() == 11) {
                    resultado = "Comprimento válido";
                } else {
                    resultado = "Comprimento inválido";
                }
                break;
            case "digitos":
                if (cpf.matches("\\d+")) {
                    resultado = "Apenas dígitos";
                } else {
                    resultado = "Contém caracteres não numéricos";
                }
                break;
            case "iguais":
                if (cpf.matches("(\\d)\\1{10}")) {
                    resultado = "Todos os dígitos iguais";
                } else {
                    resultado = "Dígitos diferentes";
                }
                break;
            case "completo":
                // Validação completa seria mais complexa
                if (cpf.length() == 11 && cpf.matches("\\d+")) {
                    resultado = "CPF válido";
                } else {
                    resultado = "CPF inválido";
                }
                break;
            default:
                resultado = "Tipo de validação não suportado";
                break;
        }

        System.out.println("CPF: " + cpf);
        System.out.println("Tipo de validação: " + tipoValidacao);
        System.out.println("Resultado: " + resultado);
    }
}
```

## Conclusão

A instrução `switch` é uma ferramenta poderosa em Java para:

- **Múltiplas condições**: Substituir longas cadeias de if-else
- **Código limpo**: Sintaxe mais clara e legível
- **Performance**: Mais eficiente que if-else para múltiplas condições
- **Manutenibilidade**: Fácil de adicionar ou remover casos

**Boas práticas**:

- Sempre use `break` para evitar fall-through
- Use `default` para casos não cobertos
- Agrupe cases relacionados
- Use switch para múltiplas condições simples
- Considere usar if-else para condições complexas

Lembre-se de escolher entre switch e if-else baseado na complexidade das condições e na legibilidade do código.
