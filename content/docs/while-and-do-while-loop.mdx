## Loops while e do...while em Java

Em programação de computadores, loops são usados para repetir um bloco de código. Por exemplo, se você quiser mostrar uma mensagem 100 vezes, você pode usar um loop. É apenas um exemplo simples; você pode conseguir muito mais com loops.

No tutorial anterior, você aprendeu sobre o [loop for em Java](./for-loop). Aqui, você vai aprender sobre os loops `while` e `do...while`.

## Loop while em Java

O loop while em Java é usado para executar um código específico até que uma certa condição seja atendida. A sintaxe do loop while é:

```java
while (expressaoTeste) {
    // corpo do loop
}
```

Aqui,

1. Um loop `while` avalia a **expressaoTeste** dentro dos parênteses `()`.
2. Se a **expressaoTeste** for avaliada como `true`, o código dentro do loop `while` é executado.
3. A **expressaoTeste** é avaliada novamente.
4. Este processo continua até que a **expressaoTeste** seja `false`.
5. Quando a **expressaoTeste** for avaliada como `false`, o loop para.

Para aprender mais sobre condições, visite [operadores relacionais Java](./operators#3-java-relational-operators) e [operadores lógicos](./operators#4-java-logical-operators).

## Fluxograma do loop while

O loop while funciona da seguinte forma:

1. **Avaliação**: Verifica a condição
2. **Execução**: Se true, executa o corpo
3. **Repetição**: Volta para a avaliação
4. **Término**: Para quando a condição é false

```
Condição → True → Corpo → Repetir
    ↓
  False → Fim
```

### Exemplo 1: Exibir Números de 1 a 5

```java
// Programa para exibir números de 1 a 5

public class Main {
    public static void main(String[] args) {
        // declarar variáveis
        int i = 1, n = 5;

        // loop while de 1 a 5
        while (i <= n) {
            System.out.println(i);
            i++;
        }
    }
}
```

#### Saída

```plaintext
1
2
3
4
5
```

Aqui está como este programa funciona.

| Iteração | Variável        | Condição: i `<=` n | Ação                                        |
| -------- | --------------- | ------------------ | ------------------------------------------- |
| 1ª       | `i = 1` `n = 5` | `true`             | `1` é impresso. `i` é aumentado para **2**. |
| 2ª       | `i = 2` `n = 5` | `true`             | `2` é impresso. `i` é aumentado para **3**. |
| 3ª       | `i = 3` `n = 5` | `true`             | `3` é impresso. `i` é aumentado para **4**. |
| 4ª       | `i = 4` `n = 5` | `true`             | `4` é impresso. `i` é aumentado para **5**. |
| 5ª       | `i = 5` `n = 5` | `true`             | `5` é impresso. `i` é aumentado para **6**. |
| 6ª       | `i = 6` `n = 5` | `false`            | O loop é terminado                          |

### Exemplo 2: Soma de Números Positivos Apenas

```java
// Programa Java para encontrar a soma de números positivos
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        int soma = 0;

        // criar um objeto da classe Scanner
        Scanner input = new Scanner(System.in);

        // receber entrada inteira do usuário
        System.out.println("Digite um número");
        int numero = input.nextInt();

        // loop while continua
        // até que o número digitado seja positivo
        while (numero >= 0) {
            // adicionar apenas números positivos
            soma += numero;

            System.out.println("Digite um número");
            numero = input.nextInt();
        }

        System.out.println("Soma = " + soma);
        input.close();
    }
}
```

## Loop do...while em Java

O loop `do...while` é similar ao loop `while`, mas há uma diferença importante: o loop `do...while` executa o corpo pelo menos uma vez antes de verificar a condição.

### Sintaxe do Loop do...while

```java
do {
    // corpo do loop
} while (expressaoTeste);
```

### Exemplo 3: Loop do...while

```java
public class DoWhileExample {
    public static void main(String[] args) {
        int i = 1, n = 5;

        // loop do...while de 1 a 5
        do {
            System.out.println(i);
            i++;
        } while (i <= n);
    }
}
```

#### Saída

```plaintext
1
2
3
4
5
```

## Diferenças entre while e do...while

| Aspecto             | Loop while                          | Loop do...while                                     |
| ------------------- | ----------------------------------- | --------------------------------------------------- |
| **Verificação**     | Verifica antes de executar          | Executa antes de verificar                          |
| **Execução mínima** | Pode não executar nenhuma vez       | Executa pelo menos uma vez                          |
| **Sintaxe**         | `while(condição) { ... }`           | `do { ... } while(condição);`                       |
| **Uso**             | Quando não sabemos se deve executar | Quando sabemos que deve executar pelo menos uma vez |

## Exemplos Práticos

### Exemplo 1: Calculadora de Média com Validação

```java
import java.util.Scanner;

public class CalculadoraMedia {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        double soma = 0;
        int contador = 0;

        System.out.println("Digite números (digite -1 para parar):");

        while (true) {
            System.out.print("Digite um número: ");
            double numero = scanner.nextDouble();

            if (numero == -1) {
                break;
            }

            soma += numero;
            contador++;
        }

        if (contador > 0) {
            double media = soma / contador;
            System.out.println("Média: " + media);
        } else {
            System.out.println("Nenhum número foi digitado.");
        }

        scanner.close();
    }
}
```

### Exemplo 2: Jogo de Adivinhação

```java
import java.util.Scanner;
import java.util.Random;

public class JogoAdivinhacao {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
```

```java
import java.util.Scanner;
```

### Exemplo 4: Validador de Senha

```java
import java.util.Scanner;

public class ValidadorSenha {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String senha;
        boolean senhaValida = false;

        do {
            System.out.print("Digite uma senha (mínimo 8 caracteres): ");
```

## Exercícios Práticos

### Exercício 1: Calculadora de Fatorial

**Objetivo**: Calcular fatorial usando loop while.

**Enunciado**: Crie um programa que calcule o fatorial de um número usando loop while.

**Resolução**:

```java
import java.util.Scanner;

public class CalculadoraFatorial {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite um número para calcular o fatorial: ");
        int numero = scanner.nextInt();

        if (numero < 0) {
            System.out.println("Fatorial não é definido para números negativos.");
        } else {
            int fatorial = 1;
            int i = 1;

            while (i <= numero) {
                fatorial *= i;
                i++;
            }

            System.out.println("Fatorial de " + numero + " = " + fatorial);
        }

        scanner.close();
    }
}
```

### Exercício 2: Série de Fibonacci

**Objetivo**: Gerar série de Fibonacci usando loop do...while.

**Enunciado**: Crie um programa que gere a série de Fibonacci usando loop do...while.

**Resolução**:

```java
import java.util.Scanner;

public class FibonacciDoWhile {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite quantos números da série Fibonacci deseja ver: ");
        int n = scanner.nextInt();

        if (n <= 0) {
            System.out.println("Digite um número positivo.");
        } else {
            int primeiro = 0, segundo = 1;
            int contador = 0;

            System.out.println("Série de Fibonacci:");

            do {
                System.out.print(primeiro + " ");

                int proximo = primeiro + segundo;
                primeiro = segundo;
                segundo = proximo;

                contador++;
            } while (contador < n);
        }

        scanner.close();
    }
}
```

### Exercício 3: Verificador de Números Primos

**Objetivo**: Verificar números primos usando loop while.

**Enunciado**: Crie um programa que verifique se um número é primo usando loop while.

**Resolução**:

```java
import java.util.Scanner;

public class VerificadorPrimos {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite um número para verificar se é primo: ");
        int numero = scanner.nextInt();

        if (numero <= 1) {
            System.out.println(numero + " não é primo.");
        } else {
            boolean ehPrimo = true;
            int divisor = 2;

            while (divisor <= Math.sqrt(numero)) {
                if (numero % divisor == 0) {
                    ehPrimo = false;
                    break;
                }
                divisor++;
            }

            if (ehPrimo) {
                System.out.println(numero + " é primo.");
            } else {
                System.out.println(numero + " não é primo.");
            }
        }

        scanner.close();
    }
}
```

## Conclusão

Os loops `while` e `do...while` são ferramentas fundamentais em Java para:

- **Iteração baseada em condição**: Executar código enquanto uma condição for verdadeira
- **Entrada de usuário**: Processar dados de entrada com validação
- **Menus interativos**: Criar interfaces de usuário simples
- **Processamento de dados**: Iterar sobre dados até uma condição ser atendida

**Boas práticas**:

- Sempre garanta que a condição do loop mude para evitar loops infinitos
- Use `break` para sair prematuramente quando necessário
- Use `continue` para pular iterações
- Escolha entre while e do...while baseado se precisa executar pelo menos uma vez
- Mantenha o código do loop simples e legível

Lembre-se de que o loop while é ideal quando você não sabe quantas iterações serão necessárias, enquanto o do...while é útil quando você sabe que pelo menos uma execução é necessária.
