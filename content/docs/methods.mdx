## O que é um Método em Java?

Em Java, um método é uma coleção de instruções projetadas para realizar tarefas específicas e, opcionalmente, retornar um resultado para o chamador. Um método Java também pode executar uma tarefa sem retornar nenhum valor. Os métodos permitem reutilização de código eliminando a necessidade de reescrever código. Importante: em Java, todo método deve pertencer a uma classe.

## Tipos de Métodos em Java

- **Métodos Definidos pelo Usuário**: Métodos personalizados criados pelo programador com base em requisitos específicos.
- **Métodos da Biblioteca Padrão**: Métodos integrados fornecidos pelo Java, prontos para uso imediato.

## Declarando um Método Java

A sintaxe básica para declaração de um método é:

```java
tipoRetorno nomeDoMetodo() {
  // corpo do método
}
```

1. **tipoRetorno**: Especifica o tipo de valor retornado pelo método. Se não retornar um valor, o tipo de retorno é `void`.
2. **nomeDoMetodo**: O identificador usado para chamar o método.
3. **corpo do método**: Código dentro de `{ }` que define a tarefa a ser executada.

### Exemplo 1:

```java
int somarNumeros() {
  // código
}
```

Neste exemplo, `somarNumeros()` é o nome do método, e seu tipo de retorno é `int`.

### Para uma declaração mais detalhada:

```java
modificador static tipoRetorno nomeDoMetodo(parametro1, parametro2, ...) {
  // corpo do método
}
```

1. **modificador**: Define o tipo de acesso (ex: public, private).
2. **static**: Se incluído, o método pode ser chamado sem criar um objeto.
3. **parametro1, parametro2, ...parametroN**: Valores passados para o método.

### Exemplo 2:

O método `sqrt()` na classe Math é estático, então pode ser acessado como `Math.sqrt()` sem criar uma instância de Math.

## Chamando um Método em Java

Para usar um método, chame-o pelo nome seguido de parênteses.

### Exemplo de Chamada de Método Java

```java
public class Main {
  public int somarNumeros(int a, int b) {
    return a + b;
  }

  public static void main(String[] args) {
    int num1 = 25, num2 = 15;
    Main obj = new Main(); // Criar objeto da classe Main
    int resultado = obj.somarNumeros(num1, num2);
    System.out.println("Soma é: " + resultado);
  }
}
```

### Saída

```bash
Soma é: 40
```

Aqui, o método `somarNumeros()` recebe dois parâmetros e retorna sua soma. Como não é estático, requer um objeto para ser chamado.

## Tipos de Retorno de Métodos

Um método pode ou não retornar um valor. A instrução `return` é usada para retornar um valor.

### Exemplo 1: Método retorna valor inteiro

```java
public static int quadrado(int num) {
  return num * num;
}
```

### Exemplo 2: Método não retorna nenhum valor

```java
// palavra-chave void é usada quando a função não retorna nenhum valor
public void imprimirQuadrado(int num) {
  System.out.println(num * num);
}
```

## Parâmetros de Métodos

Os parâmetros são valores que são passados para um método quando ele é chamado. Eles permitem que o método trabalhe com dados específicos.

### Exemplo 3: Método com múltiplos parâmetros

```java
public static double calcularMedia(double num1, double num2, double num3) {
  return (num1 + num2 + num3) / 3.0;
}
```

### 2. Métodos de Instância

```java
public class Pessoa {
    private String nome;
    private int idade;

    public Pessoa(String nome, int idade) {
        this.nome = nome;
        this.idade = idade;
    }

    public void apresentar() {
        System.out.println("Olá, sou " + nome + " e tenho " + idade + " anos.");
    }

    public boolean ehMaiorDeIdade() {
        return idade >= 18;
    }

    public void fazerAniversario() {
        idade++;
        System.out.println("Feliz aniversário! Agora tenho " + idade + " anos.");
    }

public class TestePessoa {
    public static void main(String[] args) {

        pessoa1.apresentar();
        System.out.println("É maior de idade: " + pessoa1.ehMaiorDeIdade());

        pessoa2.apresentar();
        pessoa2.fazerAniversario();
    }
}
```

### 3. Métodos com Sobrecarga (Overloading)

```java
public class CalculadoraAvancada {
    // Método para somar dois inteiros
    public static int somar(int a, int b) {
        return a + b;
    }

    // Método para somar três inteiros
    public static int somar(int a, int b, int c) {
        return a + b + c;
    }

    // Método para somar dois doubles
    public static double somar(double a, double b) {
        return a + b;
    }

    // Método para somar array de inteiros
    public static int somar(int[] numeros) {
        int soma = 0;
        for (int num : numeros) {
            soma += num;
        }
        return soma;
    }
}

public class TesteCalculadoraAvancada {
    public static void main(String[] args) {
        System.out.println("Soma de 2 inteiros: " + CalculadoraAvancada.somar(5, 3));
        System.out.println("Soma de 3 inteiros: " + CalculadoraAvancada.somar(5, 3, 2));
        System.out.println("Soma de 2 doubles: " + CalculadoraAvancada.somar(5.5, 3.3));

        int[] numeros = {1, 2, 3, 4, 5};
        System.out.println("Soma do array: " + CalculadoraAvancada.somar(numeros));
    }
}
```

### Exemplo 1: Validador de Dados

```java
public class Validador {
    public static boolean ehNumeroPositivo(int numero) {
        return numero > 0;
    }

    public static boolean ehEmailValido(String email) {
        return email != null && email.contains("@") && email.contains(".");
    }

    public static boolean ehSenhaForte(String senha) {
        if (senha == null || senha.length() < 8) {
            return false;
        }

        boolean temMaiuscula = false;
        boolean temMinuscula = false;
        boolean temNumero = false;

        for (char c : senha.toCharArray()) {
            if (Character.isUpperCase(c)) temMaiuscula = true;
            if (Character.isLowerCase(c)) temMinuscula = true;
            if (Character.isDigit(c)) temNumero = true;
        }

        return temMaiuscula && temMinuscula && temNumero;
    }

    public static void validarDados(String email, String senha, int idade) {
        System.out.println("=== Validação de Dados ===");
        System.out.println("Email válido: " + ehEmailValido(email));
        System.out.println("Senha forte: " + ehSenhaForte(senha));
        System.out.println("Idade positiva: " + ehNumeroPositivo(idade));
    }
}

public class TesteValidador {
    public static void main(String[] args) {
        Validador.validarDados("usuario@email.com", "Senha123", 25);
        System.out.println();
        Validador.validarDados("email_invalido", "123", -5);
    }
}
```

### Exemplo 4: Calculadora de Estatísticas

```java
public class CalculadoraEstatisticas {
    public static double calcularMedia(double[] numeros) {
        if (numeros.length == 0) return 0;

        double soma = 0;
        for (double num : numeros) {
            soma += num;
        }
        return soma / numeros.length;
    }

    public static double encontrarMaximo(double[] numeros) {
        if (numeros.length == 0) return 0;

        double maximo = numeros[0];
        for (double num : numeros) {
            if (num > maximo) {
                maximo = num;
            }
        }
        return maximo;
    }

    public static double encontrarMinimo(double[] numeros) {
        if (numeros.length == 0) return 0;

        double minimo = numeros[0];
        for (double num : numeros) {
            if (num < minimo) {
                minimo = num;
            }
        }
        return minimo;
    }

    public static double calcularDesvioPadrao(double[] numeros) {
        if (numeros.length == 0) return 0;

        double media = calcularMedia(numeros);
        double somaQuadrados = 0;

        for (double num : numeros) {
            somaQuadrados += Math.pow(num - media, 2);
        }

        return Math.sqrt(somaQuadrados / numeros.length);
    }

    public static void exibirEstatisticas(double[] numeros) {
        System.out.println("=== Estatísticas ===");
        System.out.println("Média: " + calcularMedia(numeros));
        System.out.println("Máximo: " + encontrarMaximo(numeros));
        System.out.println("Mínimo: " + encontrarMinimo(numeros));
        System.out.println("Desvio Padrão: " + calcularDesvioPadrao(numeros));
    }
}

public class TesteEstatisticas {
    public static void main(String[] args) {
        double[] dados = {85.5, 92.0, 78.3, 96.7, 88.2, 91.1, 82.5};
        CalculadoraEstatisticas.exibirEstatisticas(dados);
    }
}
```

### Exercício 1: Verificador de Palíndromo

**Objetivo**: Criar métodos para verificar palíndromos.

**Enunciado**: Crie métodos para verificar se uma string é palíndromo.

**Resolução**:

```java
public class VerificadorPalindromo {
    public static boolean ehPalindromo(String texto) {
        if (texto == null) return false;

        // Remover espaços e converter para minúsculas
        String limpo = texto.replaceAll("\\s", "").toLowerCase();

        int inicio = 0;
        int fim = limpo.length() - 1;

        while (inicio < fim) {
            if (limpo.charAt(inicio) != limpo.charAt(fim)) {
                return false;
            }
            inicio++;
            fim--;
        }
        return true;
    }

    public static boolean ehPalindromoRecursivo(String texto) {
        if (texto == null) return false;

        String limpo = texto.replaceAll("\\s", "").toLowerCase();
        return ehPalindromoRecursivoAux(limpo, 0, limpo.length() - 1);
    }

    private static boolean ehPalindromoRecursivoAux(String texto, int inicio, int fim) {
        if (inicio >= fim) return true;

        if (texto.charAt(inicio) != texto.charAt(fim)) {
            return false;
        }

        return ehPalindromoRecursivoAux(texto, inicio + 1, fim - 1);
    }

    public static void testarPalindromos() {
        String[] testes = {"arara", "ovo", "radar", "java", "ana", "casa"};

        System.out.println("=== Teste de Palíndromos ===");
        for (String teste : testes) {
            boolean resultado = ehPalindromo(teste);
            System.out.println("'" + teste + "' é palíndromo: " + resultado);
        }
    }
}

public class TestePalindromo {
    public static void main(String[] args) {
        VerificadorPalindromo.testarPalindromos();
    }
}
```

### Exercício 2: Calculadora de Fatorial

**Objetivo**: Criar métodos para calcular fatorial.

**Enunciado**: Crie métodos para calcular fatorial de forma iterativa e recursiva.

**Resolução**:

```java
public class CalculadoraFatorial {
    public static long fatorialIterativo(int n) {
        if (n < 0) return -1; // Erro
        if (n == 0 || n == 1) return 1;

        long resultado = 1;
        for (int i = 2; i <= n; i++) {
            resultado *= i;
        }
        return resultado;
    }

    public static long fatorialRecursivo(int n) {
        if (n < 0) return -1; // Erro
        if (n == 0 || n == 1) return 1;

        return n * fatorialRecursivo(n - 1);
    }

    public static void compararMetodos(int n) {
        System.out.println("=== Comparação de Métodos ===");
        System.out.println("Calculando fatorial de " + n);

        long inicio = System.currentTimeMillis();
        long resultadoIterativo = fatorialIterativo(n);
        long tempoIterativo = System.currentTimeMillis() - inicio;

        inicio = System.currentTimeMillis();
        long resultadoRecursivo = fatorialRecursivo(n);
        long tempoRecursivo = System.currentTimeMillis() - inicio;

        System.out.println("Resultado Iterativo: " + resultadoIterativo);
        System.out.println("Tempo Iterativo: " + tempoIterativo + "ms");
        System.out.println("Resultado Recursivo: " + resultadoRecursivo);
        System.out.println("Tempo Recursivo: " + tempoRecursivo + "ms");
    }
}

public class TesteFatorial {
    public static void main(String[] args) {
        CalculadoraFatorial.compararMetodos(10);
    }
}
```

### Exercício 3: Verificador de Números Primos

**Objetivo**: Criar métodos para verificar números primos.

**Enunciado**: Crie métodos para verificar se um número é primo.

**Resolução**:

```java
public class VerificadorPrimos {
    public static boolean ehPrimo(int numero) {
        if (numero <= 1) return false;
        if (numero <= 3) return true;
        if (numero % 2 == 0 || numero % 3 == 0) return false;

        for (int i = 5; i * i <= numero; i += 6) {
            if (numero % i == 0 || numero % (i + 2) == 0) {
                return false;
            }
        }
        return true;
    }

    public static int contarPrimos(int inicio, int fim) {
        int contador = 0;
        for (int i = inicio; i <= fim; i++) {
            if (ehPrimo(i)) {
                contador++;
            }
        }
        return contador;
    }

    public static void listarPrimos(int inicio, int fim) {
        System.out.println("Números primos entre " + inicio + " e " + fim + ":");
        for (int i = inicio; i <= fim; i++) {
            if (ehPrimo(i)) {
                System.out.print(i + " ");
            }
        }
        System.out.println();
    }

    public static void analisarPrimos(int maximo) {
        System.out.println("=== Análise de Números Primos ===");
        System.out.println("Verificando números de 1 a " + maximo);

        int totalPrimos = contarPrimos(1, maximo);
        System.out.println("Total de primos encontrados: " + totalPrimos);

        listarPrimos(1, maximo);
    }
}

public class TestePrimos {
    public static void main(String[] args) {
        VerificadorPrimos.analisarPrimos(50);
    }
}
```

### Exercício 4: Gerador de Senhas

**Objetivo**: Criar métodos para gerar senhas seguras.

**Enunciado**: Crie métodos para gerar senhas com diferentes critérios.

**Resolução**:

```java
import java.util.Random;

public class GeradorSenhas {
    private static final String LETRAS_MAIUSCULAS = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
    private static final String LETRAS_MINUSCULAS = "abcdefghijklmnopqrstuvwxyz";
    private static final String NUMEROS = "0123456789";
    private static final String CARACTERES_ESPECIAIS = "!@#$%^&*()_+-=[]{}|;:,.<>?";

    public static String gerarSenhaSimples(int tamanho) {
        String caracteres = LETRAS_MINUSCULAS + NUMEROS;
        return gerarSenha(caracteres, tamanho);
    }

    public static String gerarSenhaForte(int tamanho) {
        String caracteres = LETRAS_MAIUSCULAS + LETRAS_MINUSCULAS + NUMEROS + CARACTERES_ESPECIAIS;
        return gerarSenha(caracteres, tamanho);
    }

    public static String gerarSenhaCompleta(int tamanho) {
        Random random = new Random();
        StringBuilder senha = new StringBuilder();

        // Garantir pelo menos um de cada tipo
        senha.append(LETRAS_MAIUSCULAS.charAt(random.nextInt(LETRAS_MAIUSCULAS.length())));
        senha.append(LETRAS_MINUSCULAS.charAt(random.nextInt(LETRAS_MINUSCULAS.length())));
        senha.append(NUMEROS.charAt(random.nextInt(NUMEROS.length())));
        senha.append(CARACTERES_ESPECIAIS.charAt(random.nextInt(CARACTERES_ESPECIAIS.length())));

        // Completar o resto
        String todosCaracteres = LETRAS_MAIUSCULAS + LETRAS_MINUSCULAS + NUMEROS + CARACTERES_ESPECIAIS;
        for (int i = 4; i < tamanho; i++) {
            senha.append(todosCaracteres.charAt(random.nextInt(todosCaracteres.length())));
        }

        // Embaralhar a senha
        return embaralharString(senha.toString());
    }

    private static String gerarSenha(String caracteres, int tamanho) {
        Random random = new Random();
        StringBuilder senha = new StringBuilder();

        for (int i = 0; i < tamanho; i++) {
            senha.append(caracteres.charAt(random.nextInt(caracteres.length())));
        }

        return senha.toString();
    }

    private static String embaralharString(String texto) {
        char[] caracteres = texto.toCharArray();
        Random random = new Random();

        for (int i = caracteres.length - 1; i > 0; i--) {
            int j = random.nextInt(i + 1);
            char temp = caracteres[i];
            caracteres[i] = caracteres[j];
            caracteres[j] = temp;
        }

        return new String(caracteres);
    }

    public static void testarGeradores() {
        System.out.println("=== Gerador de Senhas ===");
        System.out.println("Senha Simples (8 chars): " + gerarSenhaSimples(8));
        System.out.println("Senha Forte (12 chars): " + gerarSenhaForte(12));
        System.out.println("Senha Completa (16 chars): " + gerarSenhaCompleta(16));
    }
}

public class TesteSenhas {
    public static void main(String[] args) {
        GeradorSenhas.testarGeradores();
    }
}
```

## Conclusão

Métodos são fundamentais na programação Java para:

- **Reutilização de código**: Evitar repetição de código
- **Organização**: Dividir problemas complexos em partes menores
- **Manutenibilidade**: Facilitar a manutenção e atualização do código
- **Testabilidade**: Facilitar testes unitários

**Boas práticas**:

- Use nomes descritivos para métodos
- Mantenha métodos pequenos e focados em uma única responsabilidade
- Use modificadores de acesso apropriados
- Documente métodos complexos
- Considere usar métodos estáticos para utilitários
- Evite métodos muito longos (máximo 20-30 linhas)

Lembre-se de que métodos bem projetados tornam o código mais legível, reutilizável e fácil de manter.
