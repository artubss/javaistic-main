**Java** é uma **linguagem de programação de propósito geral** poderosa. É usada para **desenvolver aplicações desktop e mobile, processamento de big data, sistemas embarcados**, e muito mais. Segundo a **Oracle**, a empresa que possui o **Java**, **Java roda em 3 bilhões de dispositivos em todo o mundo**, o que torna Java uma das linguagens de programação mais populares.

Para começar com programação Java, visite [Tutoriais Java](/programs/introduction).

## Características da Programação Java

#### 1. Java é independente de plataforma

Java foi construído com a filosofia de "escreva uma vez, execute em qualquer lugar" (WORA). O código Java que você escreve em uma plataforma (sistema operacional) será executado em outras plataformas sem modificação.

#### 2. Uma Linguagem Orientada a Objetos

A abordagem orientada a objetos é um dos estilos de programação populares. Na programação orientada a objetos, um problema complexo é dividido em conjuntos menores criando objetos. Isso torna o código Java reutilizável, tem benefícios de design e torna o código mais fácil de manter.

#### 3. Java é rápido

As versões anteriores do Java foram criticadas por serem lentas. No entanto, a nova versão do Java é uma das linguagens de programação mais rápidas.

Um código Java bem otimizado é quase tão rápido quanto linguagens de baixo nível como C/C++ e muito mais rápido que Python, PHP, etc.

#### 4. Java é Robusto

Robusto simplesmente significa forte. Java é robusto porque:

- Usa gerenciamento de memória forte.
- Não há ponteiros que evitam problemas de segurança.
- Há coleta automática de lixo em Java que roda na Máquina Virtual Java para se livrar de objetos que não estão mais sendo usados por uma aplicação Java.
- Há tratamento de exceções e mecanismo de verificação de tipos em Java. Todos esses pontos tornam Java robusto.

#### 5. Java é seguro

Algumas das características de alto nível para segurança que Java gerencia são:

- fornece uma plataforma segura para desenvolver e executar aplicações
- gerenciamento automático de memória reduz corrupção de memória e vulnerabilidades

#### 6. Grande Biblioteca Padrão

Uma das razões pelas quais Java é amplamente usado é devido à disponibilidade de uma enorme biblioteca padrão. O ambiente Java tem centenas de classes e métodos sob diferentes pacotes para ajudar desenvolvedores de software como nós. Por exemplo,

- `java.lang` - para recursos avançados de strings, arrays, etc
- `java.util` - para estruturas de dados, expressões regulares, funções de data e hora, etc
- `java.io` - para entrada/saída de arquivos, tratamento de exceções, etc

## Aplicações da Programação Java

Segundo a Oracle, a empresa que possui o Java, Java roda em 3 bilhões de dispositivos em todo o mundo, o que torna Java uma das linguagens de programação mais populares.

#### 1. Aplicativos Android

A linguagem de programação Java usando Android SDK (Software Development Kit) é geralmente usada para desenvolver aplicativos Android.

#### 2. Aplicações Web

Java é usado para criar aplicações web através de Servlets, Struts ou JSPs. Algumas das aplicações web populares escritas em Java são Google.com, Facebook.com, eBay.com, LinkedIn.com, etc.

#### 3. Processamento de Big Data

Você pode usar o framework de software popular como Hadoop (que é escrito em Java) para processar Big Data.

#### 4. Dispositivos Embarcados

As tecnologias Java Embedded da Oracle fornecem plataforma e runtime para bilhões de dispositivos embarcados como televisões, cartões SIM, players de Blu-ray Disc, etc.

Além dessas aplicações, Java também é usado para desenvolvimento de jogos, aplicações científicas (como processamento de linguagem natural), e muitos outros.

## Como você pode aprender a programar em Java?

#### 1. Aprenda Java com Javaistic

**Javaistic** oferece uma série completa de tutoriais Java fáceis de seguir junto com exemplos
apropriados. Estes tutoriais são direcionados para iniciantes absolutos sem conhecimento
prévio da linguagem de programação Java.

#### 2. Aprenda Java com Livros

É sempre uma boa ideia aprender a programar com livros. Você obterá a visão geral dos conceitos de programação no livro que pode não encontrar em outro lugar.

Aqui estão alguns livros que recomendamos pessoalmente.

- [**Java: The Complete Reference, Ninth Edition**](http://amzn.to/2mAdN34) - cobre tudo sobre Java, incluindo sintaxe, palavras-chave, e muito mais
- [**Thinking in Java (4th Edition)**](http://amzn.to/2mkvCCK) - melhor se você está mudando de outras linguagens de programação
- [**Java 8 Pocket Guide: Instant Help for Java Programmers**](http://amzn.to/2mEszGG) - melhor para obter respostas rápidas

#### 3. Aprenda da Documentação Oficial Java

A Oracle, a empresa que possui o Java, fornece documentação Java de qualidade. A documentação oficial é continuamente atualizada. No entanto, pode não ser o mais fácil dos tutoriais para seguir para iniciantes.

## Melhores Práticas de Programação Java

Você deve estar ansioso para aprender Java agora. No entanto, aqui estão algumas dicas e melhores práticas para seguir antes de aprender Java.

- <b>Não leia tutoriais e exemplos Java como um romance</b> : A única maneira de
  melhorar na programação é escrevendo muito código.
- <b>Aprenda a linguagem Java da maneira certa</b> : Se você está mudando de outra
  linguagem de programação (digamos C#), não escreva código no estilo C# em Java.
  Também, verifique este artigo sobre <a href="http://www.ibm.com/developerworks/library/j-perry-writing-good-java-code/index.html">Como escrever bom código Java?</a>
- <b>Junte-se a comunidades Java</b>: Uma vez que você pegue o jeito de escrever
  programas Java simples, junte-se a comunidades e fóruns Java. Então, tente
  resolver problemas Java de outros programadores. É uma boa maneira de expandir
  seu conhecimento Java. Também, você pode obter ajuda quando estiver preso.

## Palavras Finais

Você não pode errar ao aprender Java. Sua ampla gama de aplicações o torna a linguagem de oportunidades e possibilidades.
