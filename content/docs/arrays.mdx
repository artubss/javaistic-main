## Arrays em Java
Um array é uma coleção de dados do mesmo tipo.

Por exemplo, se queremos armazenar os nomes de 100 pessoas, podemos criar um array do tipo String que pode armazenar 100 nomes.

```java
String[] array = new String[100];
```
Aqui, o array acima não pode armazenar mais de 100 nomes. O número de valores em um array Java é sempre fixo.

### Como declarar um array em Java?
Em Java, aqui está como podemos declarar um array.

```java
tipoDeDados[] nomeDoArray;
```
- `tipoDeDados` - pode ser tipos primitivos como `int`, `char`, `double`, `byte`, etc. ou objetos Java
- `nomeDoArray` - é um identificador

Por exemplo,

```java
double[] dados;
```
Aqui, dados é um array que pode conter valores do tipo double.

Mas, quantos elementos este array pode conter?

Boa pergunta! Para definir o número de elementos que um array pode conter, temos que alocar memória para o array em Java. Por exemplo,

```java
// declarar um array
double[] dados;

// alocar memória
dados = new double[10];
```
Aqui, o array pode armazenar 10 elementos. Também podemos dizer que o tamanho ou comprimento do array é 10.

Em Java, podemos declarar e alocar a memória de um array em uma única declaração. Por exemplo,

```java
double[] dados = new double[10];
```

### Como Inicializar Arrays em Java?
Em Java, podemos inicializar arrays durante a declaração. Por exemplo,

```java
// declarar e inicializar um array
int[] idade = {12, 4, 5, 2, 5};
```
Aqui, criamos um array chamado idade e o inicializamos com os valores dentro das chaves.

Observe que não fornecemos o tamanho do array. Neste caso, o compilador Java especifica automaticamente o tamanho contando o número de elementos no array (ou seja, 5).

No array Java, cada localização de memória está associada a um número. O número é conhecido como índice do array. Também podemos inicializar arrays em Java usando o número do índice. Por exemplo,

```java
// declarar um array
int[] idade = new int[5];

// inicializar array
idade[0] = 12;
idade[1] = 4;
idade[2] = 5;
// ...
```
