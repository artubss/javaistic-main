## Arrays Multidimensionais em Java

Antes de aprendermos sobre arrays multidimensionais, certifique-se de que você conhece [arrays em Java](/docs/arrays).

Um array multidimensional é um array de arrays. Cada elemento de um array multidimensional é um array em si. Por exemplo,

```java
int[][] a = new int[3][4];
```

Aqui, criamos um array multidimensional chamado `a`. É um array 2-dimensional que pode conter um máximo de 12 elementos.

## Array 2-Dimensional

Lembre-se, Java usa indexação baseada em zero, ou seja, a indexação de arrays em Java começa com 0 e não 1.

Vamos pegar outro exemplo de array multidimensional. Desta vez criaremos um array 3-dimensional. Por exemplo,

```java
String[][][] data = new String[3][4][2];
```

Aqui, `data` é um array 3D que pode conter um máximo de 24 (3*4*2) elementos do tipo [String](/docs/string).

## Como inicializar um array 2D em Java?

Aqui está como podemos inicializar um array 2-dimensional em Java.

```java
int[][] a = {
    {1, 2, 3},
    {4, 5, 6, 9},
    {7},
};
```

Como podemos ver, cada elemento do array multidimensional é um array em si. E também, diferente de C/C++, cada linha do array multidimensional em Java pode ter comprimentos diferentes.

### Exemplo: Array 2-Dimensional

```java
public class MultidimensionalArray {
    public static void main(String[] args) {
        // criar um array 2d
        int[][] a = {
            {1, 2, 3},
            {4, 5, 6, 9},
            {7},
        };

        // calcular o comprimento de cada linha
        System.out.println("Comprimento da linha 1: " + a[0].length);
        System.out.println("Comprimento da linha 2: " + a[1].length);
        System.out.println("Comprimento da linha 3: " + a[2].length);
    }
}
```

#### Saída:

```plaintext
Comprimento da linha 1: 3
Comprimento da linha 2: 4
Comprimento da linha 3: 1
```

No exemplo acima, estamos criando um array multidimensional chamado `a`. Como cada componente de um array multidimensional também é um array (`a[0]`, `a[1]` e `a[2]` também são arrays).

Aqui, estamos usando o atributo `length` para calcular o comprimento de cada linha.

### Exemplo: Imprimir todos os elementos do array 2D usando Loop

```java
public class MultidimensionalArray {
    public static void main(String[] args) {
        int[][] a = {
            {1, -2, 3},
            {-4, -5, 6, 9},
            {7},
        };

        for (int i = 0; i < a.length; ++i) {
            for (int j = 0; j < a[i].length; ++j) {
                System.out.println(a[i][j]);
            }
        }
    }
}
```

#### Saída:

```plaintext
1
-2
3
-4
-5
6
9
7
```

## Tipos de Arrays Multidimensionais

### 1. Arrays 2D (Matrizes)

```java
public class Arrays2D {
    public static void main(String[] args) {
        // Matriz 3x3
        int[][] matriz = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        // Imprimir matriz
        for (int i = 0; i < matriz.length; i++) {
            for (int j = 0; j < matriz[i].length; j++) {
                System.out.print(matriz[i][j] + " ");
            }
            System.out.println();
        }
    }
}
```

## Exemplos Práticos

### Exemplo 1: Tabuleiro de Xadrez

```java
public class TabuleiroXadrez {
    public static void main(String[] args) {
        char[][] tabuleiro = new char[8][8];

        // Preencher tabuleiro
        for (int i = 0; i < 8; i++) {
            for (int j = 0; j < 8; j++) {
                if ((i + j) % 2 == 0) {
                    tabuleiro[i][j] = '□'; // Casa branca
                } else {
                    tabuleiro[i][j] = '■'; // Casa preta
                }
            }
        }

        // Imprimir tabuleiro
        System.out.println("Tabuleiro de Xadrez:");
        for (int i = 0; i < 8; i++) {
            for (int j = 0; j < 8; j++) {
                System.out.print(tabuleiro[i][j] + " ");
            }
            System.out.println();
        }
    }
}
```

### Exemplo 2: Matriz de Notas

```java
public class MatrizNotas {
    public static void main(String[] args) {
        // Matriz de notas: 3 alunos, 4 disciplinas
        double[][] notas = {
            {8.5, 7.2, 9.0, 6.8}, // Aluno 1
            {7.8, 8.9, 7.5, 8.2}, // Aluno 2
            {9.2, 8.7, 9.5, 8.9}  // Aluno 3
        };

        // Calcular média de cada aluno
        for (int i = 0; i < notas.length; i++) {
            double soma = 0;
            for (int j = 0; j < notas[i].length; j++) {
                soma += notas[i][j];
            }
            double media = soma / notas[i].length;
            System.out.println("Média do aluno " + (i + 1) + ": " + media);
        }

        // Calcular média de cada disciplina
        for (int j = 0; j < notas[0].length; j++) {
            double soma = 0;
            for (int i = 0; i < notas.length; i++) {
                soma += notas[i][j];
            }
            double media = soma / notas.length;
            System.out.println("Média da disciplina " + (j + 1) + ": " + media);
        }
    }
}
```

### Exemplo 4: Matriz de Temperaturas

```java
public class MatrizTemperaturas {
    public static void main(String[] args) {
        // Matriz de temperaturas: 7 dias, 24 horas
        double[][] temperaturas = new double[7][24];

        // Simular dados de temperatura
        for (int dia = 0; dia < 7; dia++) {
            for (int hora = 0; hora < 24; hora++) {
                // Simular temperatura baseada na hora do dia
                double tempBase = 20.0; // temperatura base
                double variacao = Math.sin(hora * Math.PI / 12) * 5.0; // variação diária
                temperaturas[dia][hora] = tempBase + variacao + (Math.random() - 0.5) * 2.0;
            }
        }

        // Encontrar temperatura máxima e mínima
        double maxTemp = temperaturas[0][0];
        double minTemp = temperaturas[0][0];

        for (int dia = 0; dia < 7; dia++) {
            for (int hora = 0; hora < 24; hora++) {
                if (temperaturas[dia][hora] > maxTemp) {
                    maxTemp = temperaturas[dia][hora];
                }
                if (temperaturas[dia][hora] < minTemp) {
                    minTemp = temperaturas[dia][hora];
                }
            }
        }

        System.out.println("Temperatura máxima: " + maxTemp + "°C");
        System.out.println("Temperatura mínima: " + minTemp + "°C");
    }
}
```

## Operações com Arrays Multidimensionais

### 1. Soma de Matrizes

```java
public class SomaMatrizes {
    public static void main(String[] args) {
        int[][] matrizA = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        int[][] matrizB = {
            {9, 8, 7},
            {6, 5, 4},
            {3, 2, 1}
        };

        int[][] resultado = new int[3][3];

        // Somar matrizes
        for (int i = 0; i < 3; i++) {
            for (int j = 0; j < 3; j++) {
                resultado[i][j] = matrizA[i][j] + matrizB[i][j];
            }
        }

        // Imprimir resultado
        System.out.println("Soma das matrizes:");
        for (int i = 0; i < 3; i++) {
            for (int j = 0; j < 3; j++) {
                System.out.print(resultado[i][j] + " ");
            }
            System.out.println();
        }
    }
}
```

```java
        }
    }
}
```

### Exercício 1: Calculadora de Média por Disciplina

**Objetivo**: Calcular médias usando arrays 2D.

**Enunciado**: Crie um programa que calcule a média de notas por disciplina usando arrays 2D.

**Resolução**:

```java
public class CalculadoraMedias {
    public static void main(String[] args) {
        // Matriz de notas: 4 alunos, 3 disciplinas
        double[][] notas = {
            {8.5, 7.2, 9.0}, // Aluno 1
            {7.8, 8.9, 7.5}, // Aluno 2
            {9.2, 8.7, 9.5}, // Aluno 3
            {6.8, 7.5, 8.2}  // Aluno 4
        };

        String[] disciplinas = {"Matemática", "Português", "História"};

        // Calcular média por disciplina
        for (int j = 0; j < notas[0].length; j++) {
            double soma = 0;
            for (int i = 0; i < notas.length; i++) {
                soma += notas[i][j];
            }
            double media = soma / notas.length;
            System.out.println("Média de " + disciplinas[j] + ": " + media);
        }

        // Calcular média por aluno
        for (int i = 0; i < notas.length; i++) {
            double soma = 0;
            for (int j = 0; j < notas[i].length; j++) {
                soma += notas[i][j];
            }
            double media = soma / notas[i].length;
            System.out.println("Média do aluno " + (i + 1) + ": " + media);
        }
    }
}
```

### Exercício 2: Verificador de Palíndromo 2D

**Objetivo**: Verificar palíndromos em arrays 2D.

**Enunciado**: Crie um programa que verifique se uma matriz 2D é um palíndromo.

**Resolução**:

```java
public class VerificadorPalindromo2D {
    public static void main(String[] args) {
        int[][] matriz = {
            {1, 2, 3, 2, 1},
            {4, 5, 6, 5, 4},
            {7, 8, 9, 8, 7},
            {4, 5, 6, 5, 4},
            {1, 2, 3, 2, 1}
        };

        boolean ehPalindromo = true;
        int linhas = matriz.length;
        int colunas = matriz[0].length;

        // Verificar se é palíndromo
        for (int i = 0; i < linhas / 2; i++) {
            for (int j = 0; j < colunas; j++) {
                if (matriz[i][j] != matriz[linhas - 1 - i][j]) {
                    ehPalindromo = false;
                    break;
                }
            }
        }

        // Verificar colunas
        for (int i = 0; i < linhas; i++) {
            for (int j = 0; j < colunas / 2; j++) {
                if (matriz[i][j] != matriz[i][colunas - 1 - j]) {
                    ehPalindromo = false;
                    break;
                }
            }
        }

        System.out.println("A matriz é palíndromo: " + ehPalindromo);
    }
}
```

### Exercício 3: Gerador de Padrões 2D

**Objetivo**: Criar padrões usando arrays 2D.

**Enunciado**: Crie um programa que gere padrões visuais usando arrays 2D.

**Resolução**:

```java
public class GeradorPadroes2D {
    public static void main(String[] args) {
        int tamanho = 5;
        char[][] padrao = new char[tamanho][tamanho];

        // Preencher com espaços
        for (int i = 0; i < tamanho; i++) {
            for (int j = 0; j < tamanho; j++) {
                padrao[i][j] = ' ';
            }
        }

        // Criar padrão de borda
        for (int i = 0; i < tamanho; i++) {
            padrao[0][i] = '*'; // Borda superior
            padrao[tamanho - 1][i] = '*'; // Borda inferior
            padrao[i][0] = '*'; // Borda esquerda
            padrao[i][tamanho - 1] = '*'; // Borda direita
        }

        // Imprimir padrão
        System.out.println("Padrão de borda:");
        for (int i = 0; i < tamanho; i++) {
            for (int j = 0; j < tamanho; j++) {
                System.out.print(padrao[i][j] + " ");
            }
            System.out.println();
        }
    }
}
```

```java
public class CalculadoraDeterminante {

            {3, 4},
            {2, 1}

        for (int i = 0; i < 2; i++) {
            for (int j = 0; j < 2; j++) {

        }

        System.out.println("Determinante: " + determinante);
    }
}
```

## Conclusão

Arrays multidimensionais são estruturas de dados poderosas em Java para:

- **Representação de dados complexos**: Matrizes, tabelas, imagens
- **Simulação de jogos**: Tabuleiros, mapas, grades
- **Processamento de dados**: Análise estatística, cálculos matemáticos
- **Organização de informações**: Dados estruturados em múltiplas dimensões

**Boas práticas**:

- Use arrays 2D para dados tabulares
- Use arrays 3D para dados volumétricos
- Sempre verifique os limites antes de acessar elementos
- Considere usar `Arrays.deepToString()` para imprimir arrays multidimensionais
- Mantenha a consistência no tamanho das dimensões quando possível

Lembre-se de que arrays multidimensionais podem ter diferentes comprimentos em cada dimensão, oferecendo flexibilidade para estruturas de dados irregulares.
