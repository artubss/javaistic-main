## Operadores em Java

Operadores são símbolos que realizam operações em variáveis e valores. Por exemplo, `+` é um operador usado para adição, enquanto `*` também é um operador usado para multiplicação.

Operadores em Java podem ser classificados em 6 tipos:

1. Operadores Aritméticos
2. Operadores de Atribuição
3. Operadores Relacionais
4. Operadores Lógicos
5. Operadores Unários
6. Operadores Bit a Bit

## 1. Operadores Aritméticos Java

Operadores aritméticos são usados para realizar operações aritméticas em variáveis e dados. Por exemplo:

```java
a + b;
```

Aqui, o operador `+` é usado para adicionar duas variáveis `a` e `b`. Da mesma forma, existem vários outros operadores aritméticos em Java.

| Operador |               Operação                |
| :------: | :-----------------------------------: |
|  **+**   |                Adição                 |
|  **-**   |               Subtração               |
|  \*\*\*  |             Multiplicação             |
|  **/**   |                Divisão                |
|  **%**   | Operação de Módulo (Resto da divisão) |

### Exemplo 1: Operadores Aritméticos

```java
public class Main {
    public static void main(String[] args) {
        // declarar variáveis
        int a = 12, b = 5;

        // operador de adição
        System.out.println("a + b = " + (a + b));

        // operador de subtração
        System.out.println("a - b = " + (a - b));

        // operador de multiplicação
        System.out.println("a * b = " + (a * b));

        // operador de divisão
        System.out.println("a / b = " + (a / b));

        // operador de módulo
        System.out.println("a % b = " + (a % b));
    }
}
```

**Saída**:

```plaintext
a + b = 17
a - b = 7
a * b = 60
a / b = 2
a % b = 2
```

No exemplo acima, usamos os operadores `+`, `-` e `*` para calcular operações de adição, subtração e multiplicação.

### Operador de Divisão /

Observe a operação `a / b` em nosso programa. O operador `/` é o operador de divisão.

Se usarmos o operador de divisão com dois inteiros, então o quociente resultante também será um inteiro. E, se um dos operandos for um número de ponto flutuante, obteremos o resultado também em ponto flutuante.

```java
Em Java,

(9 / 2) é 4
(9.0 / 2) é 4.5
(9 / 2.0) é 4.5
(9.0 / 2.0) é 4.5
```

### Operador de Módulo %

O operador de módulo `%` calcula o resto. Quando `a = 7` é dividido por `b = 4`, o resto é **3**.

<Callout>

**Nota**: O operador `%` é usado principalmente com inteiros.

</Callout>

### Exemplos Práticos de Operadores Aritméticos

```java
public class ExemplosAritmeticos {
    public static void main(String[] args) {
        // Operações básicas
        int x = 10;
        int y = 3;

        System.out.println("x = " + x + ", y = " + y);
        System.out.println("Adição: " + (x + y));
        System.out.println("Subtração: " + (x - y));
        System.out.println("Multiplicação: " + (x * y));
        System.out.println("Divisão inteira: " + (x / y));
        System.out.println("Módulo: " + (x % y));

        // Divisão com ponto flutuante
        double a = 10.0;
        double b = 3.0;
        System.out.println("Divisão decimal: " + (a / b));

        // Cálculo de área
        double largura = 5.5;
        double altura = 3.2;
        double area = largura * altura;
        System.out.println("Área do retângulo: " + area);

        // Cálculo de média
        int nota1 = 8;
        int nota2 = 7;
        int nota3 = 9;
        double media = (nota1 + nota2 + nota3) / 3.0;
        System.out.println("Média: " + media);
    }
}
```

## 2. Operadores de Atribuição Java

Operadores de atribuição são usados para atribuir valores a variáveis. Por exemplo:

```java
int age = 5;
```

Aqui, `=` é o operador de atribuição. Ele atribui o valor à direita à variável à esquerda.

### Operadores de Atribuição Compostos

| Operador | Exemplo  | Equivalente |
| :------: | :------- | :---------- |
|   `+=`   | `a += b` | `a = a + b` |
|   `-=`   | `a -= b` | `a = a - b` |
|   `*=`   | `a *= b` | `a = a * b` |
|   `/=`   | `a /= b` | `a = a / b` |
|   `%=`   | `a %= b` | `a = a % b` |

### Exemplo 2: Operadores de Atribuição

```java
public class OperadoresAtribuicao {
    public static void main(String[] args) {
        // Atribuição simples
        int a = 5;
        System.out.println("a = " + a);

        // Atribuição composta
        a += 3; // a = a + 3
        System.out.println("a += 3: " + a);

        a -= 2; // a = a - 2
        System.out.println("a -= 2: " + a);

        a *= 4; // a = a * 4
        System.out.println("a *= 4: " + a);

        a /= 2; // a = a / 2
        System.out.println("a /= 2: " + a);

        a %= 3; // a = a % 3
        System.out.println("a %= 3: " + a);
    }
}
```

## 3. Operadores Relacionais Java

Operadores relacionais são usados para verificar a relação entre dois operandos. Eles retornam `true` ou `false`.

| Operador | Descrição        | Exemplo                  |
| :------: | :--------------- | :----------------------- |
|   `==`   | Igual a          | `5 == 3` retorna `false` |
|   `!=`   | Diferente de     | `5 != 3` retorna `true`  |
|   `>`    | Maior que        | `5 > 3` retorna `true`   |
|   `<`    | Menor que        | `5 < 3` retorna `false`  |
|   `>=`   | Maior ou igual a | `5 >= 3` retorna `true`  |
|   `<=`   | Menor ou igual a | `5 <= 3` retorna `false` |

### Exemplo 3: Operadores Relacionais

```java
public class OperadoresRelacionais {
    public static void main(String[] args) {
        int a = 10;
        int b = 5;

        System.out.println("a = " + a + ", b = " + b);
        System.out.println("a == b: " + (a == b));
        System.out.println("a != b: " + (a != b));
        System.out.println("a > b: " + (a > b));
        System.out.println("a < b: " + (a < b));
        System.out.println("a >= b: " + (a >= b));
        System.out.println("a <= b: " + (a <= b));

        // Comparação de strings
        String str1 = "Java";
        String str2 = "Java";
        String str3 = "Python";

        System.out.println("str1 == str2: " + (str1 == str2));
        System.out.println("str1 == str3: " + (str1 == str3));
        System.out.println("str1.equals(str2): " + str1.equals(str2));
    }
}
```

## 4. Operadores Lógicos Java

Operadores lógicos são usados para verificar se uma expressão é `true` ou `false`. Eles são usados para tomar decisões.

| Operador | Descrição  | Exemplo                          |
| :------: | :--------- | :------------------------------- |
|   `&&`   | AND lógico | `true && false` retorna `false`  |
|  `\|\|`  | OR lógico  | `true \|\| false` retorna `true` |
|   `!`    | NOT lógico | `!true` retorna `false`          |

### Exemplo 4: Operadores Lógicos

```java
public class OperadoresLogicos {
    public static void main(String[] args) {
        boolean a = true;
        boolean b = false;

        System.out.println("a = " + a + ", b = " + b);
        System.out.println("a && b: " + (a && b));
        System.out.println("a || b: " + (a || b));
        System.out.println("!a: " + (!a));
        System.out.println("!b: " + (!b));

        // Exemplo prático
        int idade = 18;
        boolean temCarteira = true;

        boolean podeDirigir = idade >= 18 && temCarteira;
        System.out.println("Pode dirigir? " + podeDirigir);

        // Verificação de senha
        String senha = "abc123";
        boolean temComprimentoMinimo = senha.length() >= 6;
        boolean temNumero = senha.matches(".*\\d.*");

        boolean senhaValida = temComprimentoMinimo && temNumero;
        System.out.println("Senha válida? " + senhaValida);
    }
}
```

## 5. Operadores Unários Java

Operadores unários são usados com apenas um operando.

| Operador | Descrição                            |
| :------: | :----------------------------------- |
|   `+`    | Unário mais (indica valor positivo)  |
|   `-`    | Unário menos (indica valor negativo) |
|   `++`   | Incremento                           |
|   `--`   | Decremento                           |
|   `!`    | NOT lógico                           |

### Exemplo 5: Operadores Unários

```java
public class OperadoresUnarios {
    public static void main(String[] args) {
        int a = 5;
        int b = 3;

        // Operadores de incremento
        System.out.println("a = " + a);
        System.out.println("++a = " + (++a)); // Pré-incremento
        System.out.println("a++ = " + (a++)); // Pós-incremento
        System.out.println("a = " + a);

        // Operadores de decremento
        System.out.println("b = " + b);
        System.out.println("--b = " + (--b)); // Pré-decremento
        System.out.println("b-- = " + (b--)); // Pós-decremento
        System.out.println("b = " + b);

        // Operador unário menos
        int c = 10;
        System.out.println("c = " + c);
        System.out.println("-c = " + (-c));

        // Operador NOT lógico
        boolean flag = true;
        System.out.println("flag = " + flag);
        System.out.println("!flag = " + (!flag));
    }
}
```

## 6. Operadores Bit a Bit Java

Operadores bit a bit realizam operações em nível de bit.

| Operador | Descrição                        |
| :------: | :------------------------------- |
|   `&`    | AND bit a bit                    |
|   `\|`   | OR bit a bit                     |
|   `^`    | XOR bit a bit                    |
|   `~`    | NOT bit a bit                    |
|   `<<`   | Deslocamento à esquerda          |
|   `>>`   | Deslocamento à direita           |
|  `>>>`   | Deslocamento à direita sem sinal |

### Exemplo 6: Operadores Bit a Bit

```java
public class OperadoresBitABit {
    public static void main(String[] args) {
        int a = 5; // 101 em binário
        int b = 3; // 011 em binário

        System.out.println("a = " + a + " (" + Integer.toBinaryString(a) + ")");
        System.out.println("b = " + b + " (" + Integer.toBinaryString(b) + ")");

        // AND bit a bit
        System.out.println("a & b = " + (a & b) + " (" + Integer.toBinaryString(a & b) + ")");

        // OR bit a bit
        System.out.println("a | b = " + (a | b) + " (" + Integer.toBinaryString(a | b) + ")");

        // XOR bit a bit
        System.out.println("a ^ b = " + (a ^ b) + " (" + Integer.toBinaryString(a ^ b) + ")");

        // NOT bit a bit
        System.out.println("~a = " + (~a) + " (" + Integer.toBinaryString(~a) + ")");

        // Deslocamento à esquerda
        System.out.println("a << 1 = " + (a << 1) + " (" + Integer.toBinaryString(a << 1) + ")");

        // Deslocamento à direita
        System.out.println("a >> 1 = " + (a >> 1) + " (" + Integer.toBinaryString(a >> 1) + ")");
    }
}
```

## Exemplos Práticos

### Exemplo 1: Calculadora Simples

```java
import java.util.Scanner;

public class CalculadoraSimples {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite o primeiro número: ");
        double num1 = scanner.nextDouble();

        System.out.print("Digite o segundo número: ");
        double num2 = scanner.nextDouble();

        System.out.print("Digite a operação (+, -, *, /, %): ");
        char operacao = scanner.next().charAt(0);

        double resultado = 0;
        boolean operacaoValida = true;

        switch (operacao) {
            case '+':
                resultado = num1 + num2;
                break;
            case '-':
                resultado = num1 - num2;
                break;
            case '*':
                resultado = num1 * num2;
                break;
            case '/':
                if (num2 != 0) {
                    resultado = num1 / num2;
                } else {
                    System.out.println("Erro: Divisão por zero!");
                    operacaoValida = false;
                }
                break;
            case '%':
                resultado = num1 % num2;
                break;
            default:
                System.out.println("Operação inválida!");
                operacaoValida = false;
        }

        if (operacaoValida) {
            System.out.printf("%.2f %c %.2f = %.2f\n", num1, operacao, num2, resultado);
        }

        scanner.close();
    }
}
```

## Exercícios Práticos

### Exercício 1: Calculadora de IMC

**Objetivo**: Usar operadores aritméticos para calcular IMC.

**Enunciado**: Crie um programa que calcule o IMC usando operadores aritméticos.

**Resolução**:

```java
import java.util.Scanner;

public class CalculadoraIMC {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite o peso (kg): ");
        double peso = scanner.nextDouble();

        System.out.print("Digite a altura (m): ");
        double altura = scanner.nextDouble();

        // Cálculo do IMC usando operadores aritméticos
        double imc = peso / (altura * altura);

        // Classificação usando operadores relacionais
        String classificacao;
        if (imc < 18.5) {
            classificacao = "Abaixo do peso";
        } else if (imc < 25) {
            classificacao = "Peso normal";
        } else if (imc < 30) {
            classificacao = "Sobrepeso";
        } else {
            classificacao = "Obesidade";
        }

        System.out.println("=== Resultado do IMC ===");
        System.out.printf("IMC: %.2f\n", imc);
        System.out.println("Classificação: " + classificacao);

        scanner.close();
    }
}
```

### Exercício 2: Validador de Senha

**Objetivo**: Usar operadores lógicos para validar senha.

**Enunciado**: Crie um programa que valide uma senha usando operadores lógicos.

**Resolução**:

```java
import java.util.Scanner;

public class ValidadorSenha {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite uma senha: ");
        String senha = scanner.nextLine();

        // Validações usando operadores lógicos
        boolean temComprimentoMinimo = senha.length() >= 8;
        boolean temMaiuscula = false;
        boolean temMinuscula = false;
        boolean temNumero = false;
        boolean temEspecial = false;

        for (char c : senha.toCharArray()) {
            temMaiuscula = temMaiuscula || Character.isUpperCase(c);
            temMinuscula = temMinuscula || Character.isLowerCase(c);
            temNumero = temNumero || Character.isDigit(c);
            temEspecial = temEspecial || !Character.isLetterOrDigit(c);
        }

        // Validação final usando AND lógico
        boolean senhaValida = temComprimentoMinimo && temMaiuscula &&
                             temMinuscula && temNumero && temEspecial;

        System.out.println("=== Validação de Senha ===");
        System.out.println("Comprimento mínimo: " + temComprimentoMinimo);
        System.out.println("Tem maiúscula: " + temMaiuscula);
        System.out.println("Tem minúscula: " + temMinuscula);
        System.out.println("Tem número: " + temNumero);
        System.out.println("Tem caractere especial: " + temEspecial);
        System.out.println("Senha válida: " + senhaValida);

        scanner.close();
    }
}
```

### Exercício 3: Calculadora de Juros Compostos

**Objetivo**: Usar operadores aritméticos para calcular juros compostos.

**Enunciado**: Crie um programa que calcule juros compostos usando operadores aritméticos.

**Resolução**:

```java
import java.util.Scanner;

public class CalculadoraJurosCompostos {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Capital inicial: R$ ");
        double capital = scanner.nextDouble();

        System.out.print("Taxa de juros (% ao mês): ");
        double taxa = scanner.nextDouble() / 100; // Converte para decimal

        System.out.print("Tempo (meses): ");
        int tempo = scanner.nextInt();

        // Cálculo usando operadores aritméticos
        double montante = capital * Math.pow(1 + taxa, tempo);
        double juros = montante - capital;
        double rentabilidade = ((montante / capital) - 1) * 100;

        // Verificações usando operadores relacionais
        boolean lucroPositivo = juros > 0;
        boolean rentabilidadeAlta = rentabilidade > 10;
        boolean investimentoValido = capital > 0 && taxa > 0 && tempo > 0;

        System.out.println("=== Cálculo de Juros Compostos ===");
        System.out.printf("Capital inicial: R$ %.2f\n", capital);
        System.out.printf("Taxa de juros: %.2f%% ao mês\n", taxa * 100);
        System.out.println("Tempo: " + tempo + " meses");
        System.out.printf("Montante final: R$ %.2f\n", montante);
        System.out.printf("Juros ganhos: R$ %.2f\n", juros);
        System.out.printf("Rentabilidade: %.2f%%\n", rentabilidade);
        System.out.println("Lucro positivo: " + lucroPositivo);
        System.out.println("Rentabilidade alta: " + rentabilidadeAlta);
        System.out.println("Investimento válido: " + investimentoValido);

        scanner.close();
    }
}
```

## Conclusão

Operadores são fundamentais em Java para:

- **Operações matemáticas**: Aritméticos para cálculos
- **Comparações**: Relacionais para verificar condições
- **Lógica**: Lógicos para combinar condições
- **Atribuições**: De atribuição para modificar variáveis
- **Manipulação de bits**: Bit a bit para operações de baixo nível

**Boas práticas**:

- Use parênteses para clarificar precedência
- Compreenda a diferença entre `==` e `equals()`
- Use operadores de atribuição composta para código mais limpo
- Teste operadores lógicos com diferentes combinações

Lembre-se de praticar com diferentes tipos de operadores para dominar esses conceitos fundamentais.
