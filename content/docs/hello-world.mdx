Um "Hello, World!" é um programa simples que exibe `Hello, World!` na tela. Como é um programa muito simples, é frequentemente usado para introduzir uma nova linguagem de programação para iniciantes.

Vamos explorar como funciona o programa Java "Hello, World!".

## Programa Java "Hello, World!"

```java title="HelloWorld.java"
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

```

```bash title="Saída"
Hello, World!
```

## Como Funciona o Programa Java "Hello, World!"?

### 1. Seu Primeiro Programa

Em Java, qualquer linha que começa com `//` é um comentário. Comentários são destinados aos usuários que leem o código para entender a intenção e funcionalidade do programa. É completamente ignorado pelo compilador Java (uma aplicação que traduz o programa Java para bytecode Java que o computador pode executar). Para saber mais, visite [Comentários Java](/docs/comments).

### 2. class HelloWorld

Em Java, toda aplicação começa com uma definição de classe. No programa, HelloWorld é o nome da classe, e a definição da classe é:

```java
class HelloWorld {
 ... .. ...
}

```

Por enquanto, apenas lembre-se de que toda aplicação Java tem uma definição de classe, e o nome da classe deve corresponder ao nome do arquivo em Java.

### 3. public static void main(String[] args)

Este é o método principal. Toda aplicação em Java deve conter o método main. O compilador Java começa a executar o código a partir do método main.

**Como funciona?** Boa pergunta. No entanto, não discutiremos isso neste artigo. Afinal, é um programa básico para introduzir a linguagem de programação Java para iniciantes. Aprenderemos o significado de `public`, `static`, `void`, e como os métodos funcionam? em capítulos posteriores.

Por enquanto, apenas lembre-se de que a função main é o ponto de entrada da sua aplicação Java, e é obrigatória em um programa Java. A assinatura do método main em Java é:

```java
public static void main(String[] args) {
... .. ...
}
```

### 4. System.out.println("Hello, World!");

O código acima é uma declaração de impressão. Ele imprime o texto `Hello, World!` na saída padrão (sua tela). O texto dentro das aspas é chamado String em Java.

Observe que a declaração de impressão está dentro da função main, que está dentro da definição da classe.

## Pontos Importantes

- Toda Aplicação Java válida deve ter uma definição de classe (que corresponde ao nome do arquivo)
- O método main deve estar dentro da definição da classe.
- O compilador executa os códigos começando da função main.

Este é um programa Java válido que não faz nada.

```java
public class HelloWorld {
    public static void main(String[] args) {
        // Escreva seu código aqui
    }
}
```

Não se preocupe se você não entender o significado de `class`, `static`, métodos, e assim por diante por enquanto. Discutiremos isso em detalhes em capítulos posteriores.
