## Instrução if...else em Java

Em programação, usamos a instrução `if..else` para executar um bloco de código entre mais de uma alternativa.

Por exemplo, atribuir notas (A, B, C) com base na porcentagem obtida por um estudante.

- se a porcentagem for acima de **90**, atribuir nota **A**
- se a porcentagem for acima de **75**, atribuir nota **B**
- se a porcentagem for acima de **65**, atribuir nota **C**

## 1. Instrução Java if (if-then)

A sintaxe de uma instrução **if-then** é:

```java
if (condicao) {
  // instruções
}
```

Aqui, `condicao` é uma expressão booleana como `idade >= 18`.

- se `condicao` for avaliada como `true`, as instruções são executadas
- se `condicao` for avaliada como `false`, as instruções são puladas

### Exemplo 1: Instrução Java if

```java
public class IfStatement {
    public static void main(String[] args) {
        int numero = 10;

        // verifica se o número é menor que 0
        if (numero < 0) {
            System.out.println("O número é negativo.");
        }

        System.out.println("Instrução fora do bloco if");
    }
}
```

**Saída**:

```plaintext
Instrução fora do bloco if
```

No programa, `numero < 0` é `false`. Portanto, o código dentro dos parênteses é **pulado**.

## 2. Instrução Java if...else (if-then-else)

A instrução `if` executa uma certa seção de código se a expressão de teste for avaliada como `true`. No entanto, se a expressão de teste for avaliada como `false`, ela não faz nada.

Neste caso, podemos usar um bloco `else` opcional. As instruções dentro do corpo do bloco `else` são executadas se a expressão de teste for avaliada como `false`. Isso é conhecido como a instrução **if-...else** em Java.

### Sintaxe da Instrução if...else

```java
if (condicao) {
    // instruções executadas se a condição for true
} else {
    // instruções executadas se a condição for false
}
```

### Exemplo 3: Instrução Java if...else

```java
public class IfElseStatement {
    public static void main(String[] args) {
        int numero = 10;

        // verifica se o número é maior que 0
        if (numero > 0) {
            System.out.println("O número é positivo.");
        } else {
            System.out.println("O número não é positivo.");
        }
    }
}
```

**Saída**:

```plaintext
O número é positivo.
```

## 3. Instrução Java if...else if...else

A instrução `if...else if...else` é usada para executar um bloco de código entre múltiplas alternativas.

### Sintaxe da Instrução if...else if...else

```java
if (condicao1) {
    // instruções executadas se condicao1 for true
} else if (condicao2) {
    // instruções executadas se condicao2 for true
} else {
    // instruções executadas se todas as condições forem false
}
```

### Exemplo 4: Instrução Java if...else if...else

```java
public class IfElseIfStatement {
    public static void main(String[] args) {
        int numero = 0;

        if (numero > 0) {
            System.out.println("O número é positivo.");
        } else if (numero < 0) {
            System.out.println("O número é negativo.");
        } else {
            System.out.println("O número é zero.");
        }
    }
}
```

**Saída**:

```plaintext
O número é zero.
```

## 4. Instrução if Aninhada

Podemos usar uma instrução `if` dentro de outra instrução `if`. Isso é conhecido como instrução if aninhada.

### Exemplo 5: Instrução if Aninhada

```java
public class NestedIfStatement {
    public static void main(String[] args) {
        int idade = 25;
        int peso = 70;

        if (idade >= 18) {
            if (peso >= 50) {
                System.out.println("Você é elegível para doar sangue.");
            } else {
                System.out.println("Você não é elegível para doar sangue.");
            }
        } else {
            System.out.println("Você deve ter pelo menos 18 anos para doar sangue.");
        }
    }
}
```

**Saída**:

```plaintext
Você é elegível para doar sangue.
```

## Exercícios Práticos

### Exercício 1: Calculadora de Desconto

**Objetivo**: Criar um sistema de desconto baseado no valor da compra.

**Enunciado**: Crie um programa que calcule o desconto baseado no valor da compra:

- 0% para compras até R$ 100
- 5% para compras entre R$ 100 e R$ 500
- 10% para compras entre R$ 500 e R$ 1000
- 15% para compras acima de R$ 1000

**Resolução**:

```java
public class CalculadoraDesconto {
    public static void main(String[] args) {
        double valorCompra = 750.0;
        double desconto = 0.0;
        double valorFinal = 0.0;

        if (valorCompra <= 100) {
            desconto = 0.0;
        } else if (valorCompra <= 500) {
            desconto = valorCompra * 0.05;
        } else if (valorCompra <= 1000) {
            desconto = valorCompra * 0.10;
        } else {
            desconto = valorCompra * 0.15;
        }

        valorFinal = valorCompra - desconto;

        System.out.println("=== Calculadora de Desconto ===");
        System.out.printf("Valor da compra: R$ %.2f\n", valorCompra);
        System.out.printf("Desconto: R$ %.2f\n", desconto);
        System.out.printf("Valor final: R$ %.2f\n", valorFinal);
    }
}
```

### Exercício 2: Verificador de Número

**Objetivo**: Classificar números como positivo, negativo, par, ímpar, etc.

**Enunciado**: Crie um programa que classifique um número em múltiplas categorias.

**Resolução**:

```java
public class VerificadorNumero {
    public static void main(String[] args) {
        int numero = 15;

        System.out.println("=== Verificador de Número ===");
        System.out.println("Número: " + numero);

        // Verificar se é positivo, negativo ou zero
        if (numero > 0) {
            System.out.println("Tipo: Positivo");
        } else if (numero < 0) {
            System.out.println("Tipo: Negativo");
        } else {
            System.out.println("Tipo: Zero");
        }

        // Verificar se é par ou ímpar
        if (numero % 2 == 0) {
            System.out.println("Paridade: Par");
        } else {
            System.out.println("Paridade: Ímpar");
        }

        // Verificar se é primo
        boolean ehPrimo = true;
        if (numero <= 1) {
            ehPrimo = false;
        } else {
            for (int i = 2; i <= Math.sqrt(numero); i++) {
                if (numero % i == 0) {
                    ehPrimo = false;
                    break;
                }
            }
        }

        if (ehPrimo) {
            System.out.println("Primo: Sim");
        } else {
            System.out.println("Primo: Não");
        }
    }
}
```

### Exercício 3: Sistema de Login

            System.out.println("Aviso: Senha muito curta!");
        }

## Conclusão

As instruções `if...else` são fundamentais em Java para:

- **Tomada de decisão**: Executar código baseado em condições
- **Controle de fluxo**: Direcionar a execução do programa
- **Validação**: Verificar dados e condições
- **Classificação**: Categorizar dados baseado em critérios

**Boas práticas**:

- Use condições claras e legíveis
- Evite aninhamento excessivo de ifs
- Use `else if` para múltiplas condições mutuamente exclusivas
- Sempre considere o caso `else` quando apropriado
- Use operadores lógicos para combinar condições

Lembre-se de praticar com diferentes cenários para dominar o controle de fluxo em Java.
