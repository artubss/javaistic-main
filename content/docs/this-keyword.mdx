## O que é a palavra-chave 'this' em Java?

Em Java, a palavra-chave `this` é usada dentro de uma classe para se referir à instância atual (objeto) dessa classe. É comumente usada em métodos ou construtores para referenciar as propriedades do objeto atual ou para chamar outros construtores dentro da mesma classe.

### Exemplo Básico

```java
public class Main {
    int instVar;

    Main(int instVar) {
        this.instVar = instVar;
        System.out.println("referência this = " + this);
    }

    public static void main(String[] args) {
        Main obj = new Main(8);
        System.out.println("referência do objeto = " + obj);
    }
}
```

**Saída**:

```bash
referência this = Main@23fc625e
referência do objeto = Main@23fc625e
```

Neste exemplo, `this` se refere à instância `obj` de Main. Tanto `this` quanto `obj` compartilham a mesma referência de memória, indicando que `this` representa o objeto atual.

## Usos Comuns da Palavra-chave this

### 1. Resolvendo Conflitos de Nomes de Variáveis

Quando variáveis de instância e parâmetros têm o mesmo nome, usar `this` ajuda a evitar ambiguidade:

```java
public class MinhaClasse {
    int idade;

    MinhaClasse(int idade) {
        this.idade = idade; // Usa `this` para diferenciar variável de instância do parâmetro
    }
}
```

**Sem `this`**:

```java
public class Main {
    int idade;
    
    Main(int idade) {
        idade = idade; // Sem `this`, então não há atribuição à variável de instância
    }

    public static void main(String[] args) {
        Main obj = new Main(8);
        System.out.println("obj.idade = " + obj.idade); // Saída: obj.idade = 0
    }
}
```

### 2. Usando this com Getters e Setters

`this` é frequentemente usado em métodos setter para diferenciar entre variáveis de instância e parâmetros:

```java
public class Main {
    String nome;

    void setNome(String nome) {
        this.nome = nome;
    }

    String getNome() {
        return this.nome;
    }

    public static void main(String[] args) {
        Main obj = new Main();
        obj.setNome("João");
        System.out.println("obj.nome: " + obj.getNome());
    }
}
```

**Saída**:

```bash
obj.nome: João
```

### 3. Sobrecarga de Construtores com this()

Na sobrecarga de construtores, `this()` pode ser usado para chamar outro construtor dentro da mesma classe, reduzindo duplicação de código:

```java
public class Complexo {
    private int a, b;

    // Construtor com dois parâmetros
    Complexo(int i, int j) {
        this.a = i;
        this.b = j;
    }

    // Construtor com um parâmetro
    Complexo(int i) {
        this(i, i); // Chama o construtor com dois parâmetros
    }

    // Construtor padrão
    Complexo() {
        this(0, 0); // Chama o construtor com dois parâmetros
    }

    public void exibir() {
        System.out.println("a = " + a + ", b = " + b);
    }
}
```

### Exemplo 1: Classe Pessoa

```java
public class Pessoa {
    private String nome;
    private int idade;
    private String email;
    
    // Construtor principal
    public Pessoa(String nome, int idade, String email) {
        this.nome = nome;
        this.idade = idade;
        this.email = email;
    }
    
    // Construtor sobrecarregado
    public Pessoa(String nome, int idade) {
        this(nome, idade, "sem email"); // Chama o construtor principal
    }
    
    // Construtor padrão
    public Pessoa() {
        this("Sem nome", 0, "sem email"); // Chama o construtor principal
    }
    
    // Getters
    public String getNome() {
        return this.nome;
    }
    
    public int getIdade() {
        return this.idade;
    }
    
    public String getEmail() {
        return this.email;
    }
    
    // Setters
    public void setNome(String nome) {
        this.nome = nome;
    }
    
    public void setIdade(int idade) {
        this.idade = idade;
    }
    
    public void setEmail(String email) {
        this.email = email;
    }
    
    // Método que usa this para chamar outro método
    public void exibirInformacoes() {
        System.out.println("Nome: " + this.getNome());
        System.out.println("Idade: " + this.getIdade());
        System.out.println("Email: " + this.getEmail());
    }
    
    // Método que retorna this para method chaining
    public Pessoa atualizarNome(String nome) {
        this.nome = nome;
        return this; // Retorna a instância atual
    }
    
    public Pessoa atualizarIdade(int idade) {
        this.idade = idade;
        return this; // Retorna a instância atual
    }
    
    public Pessoa atualizarEmail(String email) {
        this.email = email;
        return this; // Retorna a instância atual
    }
}
```

### Exemplo 2: Classe Retângulo

```java
public class Retangulo {
    private double largura;
    private double altura;
    
    // Construtor principal
    public Retangulo(double largura, double altura) {
        this.largura = largura;
        this.altura = altura;
    }
    
    // Construtor para quadrado
    public Retangulo(double lado) {
        this(lado, lado); // Chama o construtor principal
    }
    
    // Construtor padrão
    public Retangulo() {
        this(1.0, 1.0); // Retângulo padrão 1x1
    }
    
    // Getters
    public double getLargura() {
        return this.largura;
    }
    
    public double getAltura() {
        return this.altura;
    }
    
    // Setters
    public void setLargura(double largura) {
        this.largura = largura;
    }
    
    public void setAltura(double altura) {
        this.altura = altura;
    }
    
    // Métodos de cálculo
    public double calcularArea() {
        return this.largura * this.altura;
    }
    
    public double calcularPerimetro() {
        return 2 * (this.largura + this.altura);
    }
    
    // Método que usa this para comparar objetos
    public boolean temMesmaArea(Retangulo outro) {
        return this.calcularArea() == outro.calcularArea();
    }
    
    // Método que retorna this para method chaining
    public Retangulo redimensionar(double fator) {
        this.largura *= fator;
        this.altura *= fator;
        return this;
    }
}
```

### Exemplo 3: Classe Conta Bancária

```java
public class ContaBancaria {
    private String numeroConta;
    private String titular;
    private double saldo;
    private String tipoConta;
    
    // Construtor principal
    public ContaBancaria(String numeroConta, String titular, double saldoInicial, String tipoConta) {
        this.numeroConta = numeroConta;
        this.titular = titular;
        this.saldo = saldoInicial;
        this.tipoConta = tipoConta;
    }
    
    // Construtor sobrecarregado
    public ContaBancaria(String numeroConta, String titular, double saldoInicial) {
        this(numeroConta, titular, saldoInicial, "Corrente");
    }
    
    // Construtor padrão
    public ContaBancaria() {
        this("000000", "Sem titular", 0.0, "Corrente");
    }
    
    // Getters
    public String getNumeroConta() {
        return this.numeroConta;
    }
    
    public String getTitular() {
        return this.titular;
    }
    
    public double getSaldo() {
        return this.saldo;
    }
    
    public String getTipoConta() {
        return this.tipoConta;
    }
    
    // Setters
    public void setTitular(String titular) {
        this.titular = titular;
    }
    
    public void setTipoConta(String tipoConta) {
        this.tipoConta = tipoConta;
    }
    
    // Métodos de operação
    public void depositar(double valor) {
        if (valor > 0) {
            this.saldo += valor;
            System.out.println("Depósito de R$ " + valor + " realizado.");
        } else {
            System.out.println("Valor inválido para depósito.");
        }
    }
    
    public boolean sacar(double valor) {
        if (valor > 0 && valor <= this.saldo) {
            this.saldo -= valor;
            System.out.println("Saque de R$ " + valor + " realizado.");
            return true;
        } else {
            System.out.println("Saldo insuficiente ou valor inválido.");
            return false;
        }
    }
    
    // Método que usa this para transferência
    public boolean transferir(ContaBancaria destino, double valor) {
        if (this.sacar(valor)) {
            destino.depositar(valor);
            System.out.println("Transferência de R$ " + valor + " realizada.");
            return true;
        }
        return false;
    }
    
    // Método que retorna this para method chaining
    public ContaBancaria atualizarTitular(String titular) {
        this.titular = titular;
        return this;
    }
    
    public ContaBancaria alterarTipoConta(String tipoConta) {
        this.tipoConta = tipoConta;
        return this;
    }
    
    // Método que exibe informações
    public void exibirInformacoes() {
        System.out.println("=== Informações da Conta ===");
        System.out.println("Número: " + this.getNumeroConta());
        System.out.println("Titular: " + this.getTitular());
        System.out.println("Saldo: R$ " + this.getSaldo());
        System.out.println("Tipo: " + this.getTipoConta());
    }
}
```

## Method Chaining com this

O `this` é muito útil para implementar method chaining (encadeamento de métodos):

```java
public class Builder {
    private String nome;
    private int idade;
    private String email;
    
    public Builder setNome(String nome) {
        this.nome = nome;
        return this; // Retorna a instância atual
    }
    
    public Builder setIdade(int idade) {
        this.idade = idade;
        return this; // Retorna a instância atual
    }
    
    public Builder setEmail(String email) {
        this.email = email;
        return this; // Retorna a instância atual
    }
    
    public Pessoa construir() {
        return new Pessoa(nome, idade, email);
    }
}

// Uso
public class Main {
    public static void main(String[] args) {
        Pessoa pessoa = new Builder()
            .setNome("João")
            .setIdade(25)
            .setEmail("joao@email.com")
            .construir();
    }
}
```

### Exercício 1: Classe Produto
**Objetivo**: Criar uma classe Produto usando this adequadamente.

**Enunciado**: Crie uma classe `Produto` com construtores sobrecarregados e method chaining.

**Resolução**:

```java
public class Produto {
    private String codigo;
    private String nome;
    private double preco;
    private int quantidade;
    
    // Construtor principal
    public Produto(String codigo, String nome, double preco, int quantidade) {
        this.codigo = codigo;
        this.nome = nome;
        this.preco = preco;
        this.quantidade = quantidade;
    }
    
    // Construtor sobrecarregado
    public Produto(String codigo, String nome, double preco) {
        this(codigo, nome, preco, 0);
    }
    
    // Construtor padrão
    public Produto() {
        this("000000", "Produto sem nome", 0.0, 0);
    }
    
    // Getters
    public String getCodigo() { return this.codigo; }
    public String getNome() { return this.nome; }
    public double getPreco() { return this.preco; }
    public int getQuantidade() { return this.quantidade; }
    
    // Setters com method chaining
    public Produto setCodigo(String codigo) {
        this.codigo = codigo;
        return this;
    }
    
    public Produto setNome(String nome) {
        this.nome = nome;
        return this;
    }
    
    public Produto setPreco(double preco) {
        this.preco = preco;
        return this;
    }
    
    public Produto setQuantidade(int quantidade) {
        this.quantidade = quantidade;
        return this;
    }
    
    // Métodos de negócio
    public double calcularValorTotal() {
        return this.preco * this.quantidade;
    }
    
    public void adicionarEstoque(int quantidade) {
        this.quantidade += quantidade;
    }
    
    public boolean removerEstoque(int quantidade) {
        if (quantidade <= this.quantidade) {
            this.quantidade -= quantidade;
            return true;
        }
        return false;
    }
    
    public void exibirInformacoes() {
        System.out.println("=== Informações do Produto ===");
        System.out.println("Código: " + this.getCodigo());
        System.out.println("Nome: " + this.getNome());
        System.out.println("Preço: R$ " + this.getPreco());
        System.out.println("Quantidade: " + this.getQuantidade());
        System.out.println("Valor Total: R$ " + this.calcularValorTotal());
    }
}

// Teste
public class Main {
    public static void main(String[] args) {
        // Usando construtores
        Produto p1 = new Produto("001", "Notebook", 2500.0, 5);
        Produto p2 = new Produto("002", "Mouse", 50.0);
        Produto p3 = new Produto();
        
        // Usando method chaining
        Produto p4 = new Produto()

            .setPreco(100.0)
            .setQuantidade(10);
```

### Exercício 2: Classe Aluno
**Objetivo**: Criar uma classe Aluno com construtores sobrecarregados.

**Enunciado**: Crie uma classe `Aluno` com diferentes construtores e métodos que usam this.

**Resolução**:

```java
public class Aluno {
    private String matricula;
    private String nome;
    private int idade;
    private String curso;
    private double[] notas;
    
    // Construtor principal
    public Aluno(String matricula, String nome, int idade, String curso, double[] notas) {
        this.matricula = matricula;
        this.nome = nome;
        this.idade = idade;
        this.curso = curso;
        this.notas = notas;
    }
    
    // Construtor sobrecarregado
    public Aluno(String matricula, String nome, int idade, String curso) {
        this(matricula, nome, idade, curso, new double[4]);
    }
    
    // Construtor padrão
    public Aluno() {
        this("000000", "Sem nome", 0, "Sem curso", new double[4]);
    }
    
    // Getters
    public String getMatricula() { return this.matricula; }
    public String getNome() { return this.nome; }
    public int getIdade() { return this.idade; }
    public String getCurso() { return this.curso; }
    public double[] getNotas() { return this.notas; }
    
    // Setters
    public void setNome(String nome) {
        this.nome = nome;
    }
    
    public void setIdade(int idade) {
        this.idade = idade;
    }
    
    public void setCurso(String curso) {
        this.curso = curso;
    }
    
    // Métodos de negócio
    public void adicionarNota(int posicao, double nota) {
        if (posicao >= 0 && posicao < this.notas.length) {
            this.notas[posicao] = nota;
        }
    }
    
    public double calcularMedia() {
        double soma = 0;
        int contador = 0;
        
        for (double nota : this.notas) {
            if (nota > 0) {
                soma += nota;
                contador++;
            }
        }
        
        return contador > 0 ? soma / contador : 0;
    }
    
    public boolean estaAprovado() {
        return this.calcularMedia() >= 7.0;
    }
    
    public void exibirInformacoes() {
        System.out.println("=== Informações do Aluno ===");
        System.out.println("Matrícula: " + this.getMatricula());
        System.out.println("Nome: " + this.getNome());
        System.out.println("Idade: " + this.getIdade());
        System.out.println("Curso: " + this.getCurso());
        System.out.println("Média: " + this.calcularMedia());
        System.out.println("Status: " + (this.estaAprovado() ? "Aprovado" : "Reprovado"));
    }
}

// Teste
public class Main {
    public static void main(String[] args) {
        // Criando alunos com diferentes construtores
        Aluno a1 = new Aluno("001", "João", 20, "Engenharia", new double[]{8.5, 7.0, 9.0, 8.0});
        Aluno a2 = new Aluno("002", "Maria", 19, "Medicina");
        Aluno a3 = new Aluno();
        
        // Adicionando notas
        a2.adicionarNota(0, 9.5);
        a2.adicionarNota(1, 8.0);
        a2.adicionarNota(2, 7.5);
        a2.adicionarNota(3, 9.0);
        
        a3.setNome("Pedro");
        a3.setIdade(21);
        a3.setCurso("Direito");
        a3.adicionarNota(0, 6.5);
        a3.adicionarNota(1, 7.0);
        
        // Exibindo informações
        a1.exibirInformacoes();
        System.out.println();
        a2.exibirInformacoes();
        System.out.println();
        a3.exibirInformacoes();
    }
```

```java
    private String titulo;
    private String autor;

    public int getAnoPublicacao() { return this.anoPublicacao; }
    public int getNumeroPaginas() { return this.numeroPaginas; }

        System.out.println("Páginas: " + this.getNumeroPaginas());

        public LivroBuilder setTitulo(String titulo) {
            this.titulo = titulo;
            return this;
        }
        
        public LivroBuilder setAutor(String autor) {
            this.autor = autor;
            return this;
        }
        
        public LivroBuilder setEditora(String editora) {
            this.editora = editora;
            return this;
        }
        
        public LivroBuilder setAnoPublicacao(int anoPublicacao) {
            this.anoPublicacao = anoPublicacao;

        }
```

## Conclusão

A palavra-chave `this` é fundamental em Java para:

- **Referenciar a instância atual**: Acessar membros da classe atual
- **Resolver conflitos de nomes**: Diferenciar variáveis de instância de parâmetros
- **Chamar construtores**: Implementar sobrecarga de construtores
- **Method chaining**: Permitir encadeamento de métodos
- **Melhorar legibilidade**: Tornar o código mais claro e expressivo

**Casos de uso comuns**:
- Construtores sobrecarregados
- Getters e setters
- Method chaining (Builder pattern)
- Comparação de objetos
- Referência a métodos da própria classe

Lembre-se de usar `this` quando necessário para evitar ambiguidade e tornar seu código mais claro e profissional.
