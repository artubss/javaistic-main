## Loop for-each em Java

Em Java, o **loop for-each** é usado para iterar através dos elementos de [arrays](/docs/arrays) e coleções (como [ArrayList](/docs/arraylist)). Também é conhecido como loop for aprimorado.

## Sintaxe do Loop `for-each`

A sintaxe do loop **for-each** em Java é:

```java
for(tipoDados item : array) {
    ...
}
```

Aqui,

- **array** - um array ou uma coleção
- **item** - cada item do array/coleção é atribuído a esta variável
- **tipoDados** - o tipo de dados do array/coleção

### Exemplo 1: Imprimir Elementos do Array

#### Entrada

```java
// imprimir elementos do array

public class Main {
    public static void main(String[] args) {
        // criar um array
        int[] numeros = {3, 9, 5, -5};

        // loop for-each
        for (int numero : numeros) {
            System.out.println(numero);
        }
    }
}
```

#### Saída

```plaintext
3
9
5
-5
```

Aqui, usamos o **loop for-each** para imprimir cada elemento do array numeros um por um.

- Na primeira iteração, o item será 3.
- Na segunda iteração, o item será 9.
- Na terceira iteração, o item será 5.
- Na quarta iteração, o item será -5.

### Exemplo 2: Soma dos Elementos do Array

#### Entrada

```java
// Calcular a soma de todos os elementos de um array

public class Main {
    public static void main(String[] args) {
        // um array de números
        int[] numeros = {3, 4, 5, -5, 0, 12};
        int soma = 0;

        // iterando através de cada elemento do array
        for (int numero : numeros) {
            soma += numero;
        }

        System.out.println("Soma = " + soma);
    }
}
```

#### Saída:

```plaintext
Soma = 19
```

No programa acima, a execução do loop for-each se parece com:

| Iteração | Variáveis                            |
| -------- | ------------------------------------ |
| 1        | `numero` = 3 `soma` = 0 + 3 = 3      |
| 2        | `numero` = 4 `soma` = 3 + 4 = 7      |
| 3        | `numero` = 5 `soma` = 7 + 5 = 12     |
| 4        | `numero` = -5 `soma` = 12 + (-5) = 7 |
| 5        | `numero` = 0 `soma` = 7 + 0 = 7      |
| 6        | `numero` = 12 `soma` = 7 + 12 = 19   |

Como podemos ver, adicionamos cada elemento do array `numeros` à variável `soma` em cada iteração do loop.

## Loop for vs Loop for-each

| Aspecto          | Loop for                                | Loop for-each                 |
| ---------------- | --------------------------------------- | ----------------------------- |
| **Sintaxe**      | `for(int i = 0; i < array.length; i++)` | `for(int item : array)`       |
| **Controle**     | Controle total sobre índice             | Sem acesso ao índice          |
| **Modificação**  | Pode modificar elementos                | Não pode modificar elementos  |
| **Legibilidade** | Mais verboso                            | Mais limpo e legível          |
| **Performance**  | Ligeiramente mais rápido                | Ligeiramente mais lento       |
| **Uso**          | Quando precisa do índice                | Quando só precisa dos valores |

### Exemplo 3: Comparação entre for e for-each

```java
public class ComparacaoLoops {
    public static void main(String[] args) {
        String[] frutas = {"maçã", "banana", "laranja", "uva"};

        // Loop for tradicional
        System.out.println("Loop for tradicional:");
        for (int i = 0; i < frutas.length; i++) {
            System.out.println("Índice " + i + ": " + frutas[i]);
        }

        // Loop for-each
        System.out.println("\nLoop for-each:");
        for (String fruta : frutas) {
            System.out.println("Fruta: " + fruta);
        }
    }
}
```

## Vantagens do Loop for-each

### 1. Código Mais Limpo

```java
// Com loop for tradicional
for (int i = 0; i < array.length; i++) {
    System.out.println(array[i]);
}

// Com loop for-each
for (int item : array) {
    System.out.println(item);
}
```

### 2. Menos Propenso a Erros

```java
// Erro comum com loop for tradicional
for (int i = 0; i <= array.length; i++) { // <= causa ArrayIndexOutOfBoundsException
    System.out.println(array[i]);
}

// Loop for-each evita esse erro
for (int item : array) {
    System.out.println(item); // Sempre seguro
}
```

### 3. Funciona com Coleções

```java
import java.util.ArrayList;
import java.util.List;

public class ExemploColecoes {

        nomes.add("Pedro");

        // Loop for-each funciona com List

    }
```

## Exemplos Práticos

### Exemplo 1: Processamento de Arrays

```java
public class ProcessamentoArrays {
    public static void main(String[] args) {
        int[] numeros = {10, 20, 30, 40, 50};

        // Encontrar o maior número
        int maior = numeros[0];
        for (int numero : numeros) {
            if (numero > maior) {
                maior = numero;
            }
        }

        System.out.println("Maior número: " + maior);

        // Contar números pares
        int pares = 0;
        for (int numero : numeros) {
            if (numero % 2 == 0) {
                pares++;
            }
        }

        System.out.println("Números pares: " + pares);
    }
}
```

```java
public class ProcessamentoStrings {
```

## Exercícios Práticos

### Exercício 1: Calculadora de Estatísticas

**Objetivo**: Calcular estatísticas de um array usando for-each.

**Enunciado**: Crie um programa que calcule média, máximo e mínimo de um array.

**Resolução**:

```java
public class CalculadoraEstatisticas {
    public static void main(String[] args) {
        double[] numeros = {85.5, 92.0, 78.3, 96.7, 88.2, 91.1};

        double soma = 0;
        double maximo = numeros[0];
        double minimo = numeros[0];

        for (double numero : numeros) {
            soma += numero;

            if (numero > maximo) {
                maximo = numero;
            }

            if (numero < minimo) {
                minimo = numero;
            }
        }

        double media = soma / numeros.length;

        System.out.println("Estatísticas:");
        System.out.println("Média: " + media);
        System.out.println("Máximo: " + maximo);
        System.out.println("Mínimo: " + minimo);
    }
}
```

### Exercício 2: Validador de Senhas

**Objetivo**: Validar uma lista de senhas usando for-each.

**Enunciado**: Crie um programa que valide uma lista de senhas.

**Resolução**:

```java
public class ValidadorSenhas {
    public static void main(String[] args) {
        String[] senhas = {"abc123", "senha123", "123456", "MinhaSenha123", "qwerty"};

        System.out.println("Validação de senhas:");

        for (String senha : senhas) {
            boolean temMaiuscula = false;
            boolean temMinuscula = false;
            boolean temNumero = false;
            boolean temComprimento = senha.length() >= 8;

            for (char caractere : senha.toCharArray()) {
                if (Character.isUpperCase(caractere)) {
                    temMaiuscula = true;
                }
                if (Character.isLowerCase(caractere)) {
                    temMinuscula = true;
                }
                if (Character.isDigit(caractere)) {
                    temNumero = true;
                }
            }

            boolean senhaValida = temMaiuscula && temMinuscula && temNumero && temComprimento;

            System.out.println("'" + senha + "': " + (senhaValida ? "Válida" : "Inválida"));
        }
    }
}
```

### Exercício 3: Processador de Texto

**Objetivo**: Processar um array de textos usando for-each.

**Enunciado**: Crie um programa que processe um array de textos.

**Resolução**:

```java
public class ProcessadorTexto {
    public static void main(String[] args) {
        String[] textos = {
            "Java é uma linguagem de programação",
            "Python é muito popular",
            "JavaScript é usado na web",
            "C++ é uma linguagem poderosa"
        };

        int totalPalavras = 0;
        String textoMaisLongo = "";

        for (String texto : textos) {
            // Contar palavras
            String[] palavras = texto.split("\\s+");
            totalPalavras += palavras.length;

            // Encontrar texto mais longo
            if (texto.length() > textoMaisLongo.length()) {
                textoMaisLongo = texto;
            }
        }

        System.out.println("Total de palavras: " + totalPalavras);
        System.out.println("Texto mais longo: " + textoMaisLongo);

        // Listar textos que contêm "Java"
        System.out.println("Textos que contêm 'Java':");
        for (String texto : textos) {
            if (texto.toLowerCase().contains("java")) {
                System.out.println("- " + texto);
            }
        }
    }
}
```

## Conclusão

O loop **for-each** é uma ferramenta poderosa em Java para:

- **Iteração simples**: Percorrer arrays e coleções sem se preocupar com índices
- **Código limpo**: Sintaxe mais legível e menos propensa a erros
- **Segurança**: Evita erros de índice fora dos limites
- **Compatibilidade**: Funciona com arrays e coleções

**Boas práticas**:

- Use for-each quando não precisa do índice
- Use for tradicional quando precisa modificar elementos ou acessar índices
- For-each é ideal para leitura de dados
- Mantenha o código simples e legível

Lembre-se de escolher o tipo de loop apropriado baseado nas suas necessidades específicas.
